HTML快速入门 新建文本文件，后缀名为.html 编写HTML的基本框架，定义标题 <html> <head> <title>HTML快速入门</title> <!-- 网页头部：用来存放给浏览器的信息， 如：CSS样式 --> </head> <body> <h1>Hello World!</h1> <img src="1.jpg"> <!-- 网页主体：用来存放给用户看的信息， 如：文字、图片、视频 --> </body> </html> 在<body>中填写内容 HTML标签的特点 html标签不区分大小写，建议小写 html标签的属性值使用单引号/双引号都可以 html语法结构松散，但是建议规范书写 标准HTML文件 <!-- 声明文档类型 --> <!DOCTYPE v-html="html"> <html lang="en"> <head> <!-- 文档类型 --> <meta charset="UTF-8" /> <!-- 解决浏览器兼容问题 --> <meta name="viewport" content="width=device-width, initial-scale=1.0" /> <!-- 标题 --> <title>HTML快速入门</title> </head> <body> <h1>Hello Html</h1> <img src="../1.jpg"> </body> </html> HTML常见标签 数据展示标签 标题标签：<h1> - <h6> <h1>一级标题</h1> <h2>二级标题</h2> <h3>三级标题</h3> 段落与文本标签 <p>: 段落 <br>: 换行 水平线标签：<hr> <strong>: 加粗，强调 <em>: 斜体，强调 <p>这是一个段落。</p> <p>这是第二段<br>带换行</p> <strong>加粗文字</strong> <em>斜体文字</em> 列表标签 <ol>：有序列表 <ul>：无序列表 li：列表项 <ol> <li>第一项</li> <li>第二项</li> </ol> <ul> <li>苹果</li> <li>香蕉</li> </ul> 链接标签：<a> 图片标签：<img> src: 指定图像的url（绝对路径/相对路径） 绝对路径 绝对磁盘路径 绝对网络路径 相对路径 . 代表当前路径 .. 代表上一级路径 alt: 替换文本，图片无法显示时展示的文字 width: 图像的宽度（像素/相对于父元素的百分比） height: 图像的高度（像素/相对于父元素的百分比） 表格标签 <table>: 表格 <tr>：表格头 <td>: 单元格 <th>: 表头 CSS样式控制 CSS的引入方式 内联样式（Inline Style） 行内样式：写在标签的style属性中 示例： <p style="color: red; font-size: 16px;">这是红色文字</p> 内部样式表（Internal Style Sheet） 写在<style>标签中（可以写在任何位置，但通常写在<head>标签中） <head> <style> p { color: blue; font-size: 18px; } h1 { color: red; } </style> </head> 外部样式表（External Style Sheet） 写在一个单独的.css文件中（需要通过link标签在网页中引入 示例： <head> <link rel="stylesheet" href="style.css"> </head> rel:描述与当前文档的关系 stylesheet: 当前文档的样式表 href:资源的URL地址 常用CSS属性 分类 常用属性 示例 文本样式 color、font-size、font-weight、text-align p { color: red;
