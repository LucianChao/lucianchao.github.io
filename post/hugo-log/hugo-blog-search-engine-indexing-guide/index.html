【建站日志】如何让自己的博客被搜索引擎收录

2025-11-21 · (最后修改于 2025-11-22) · 3 分钟

Bing Webmasster Tools是必应的网站管理员工具，可以添加个人网站并带进行一些网站操作。

  1. 打开Bing Webmasster Tools

    点击"开始"或右上角"登录"以登录到官网。可以选择三种登录账号：Google、Microsoft、Facebook。这里我用 Microsoft 账号登录。

  2. 初次登录会提示你添加网站，有两种添加方式：从 GSC 导入、手动添加。如果您的网站被 谷歌GSC 收录，可以用 GSC 直接导入。

    这里我选择手动添加，输入自己的域名，点击添加（在笔者查阅资料时，有人说这里可能会出现网络问题，可以多试几次，不过我并没有遇到此类问题，或许是因为当时开启了科学上网工具）

  3. 现在需要你验证网站的所有权，即验证这个网站是属于你的。

    这里我选择第一种方式，使用XML文件验证

  4. 下载XML文件，放在网站静态资源目录下：/static/BingSiteAuth.xml，重新生成网站并推送到远程仓库中，可以检查远程仓库根目录中是否有BingSiteAuth.xml

  5. 点击验证，添加成功

URL提交 和 网站地图#

简单说一下搜索引擎是如何工作的（这里只做基础介绍，实际原理要比这复杂得多）。

像百度、谷歌、必应这些搜索引擎，都会维护一个巨大的数据库，里面记录着各种网页的 URL、关键词甚至服务器信息。用户输入搜索词后，搜索引擎就在这个数据库里查找匹配内容，并把结果展示出来，用户点一下链接就能访问网站。

那这些数据是怎么进入搜索引擎的呢？主要有两条路径：

  • 人工提交；
  • 搜索引擎用自己的爬虫自动抓取。

一般来说，搜索引擎会优先抓取一些大型或权重高的网站。

但对我们这种刚建好的“小破站”来说，爬虫往往不会马上来，所以需要我们主动“通知”搜索引擎一下。常用方式有三种：

  1. 把自己的网站地址放在已被收录的页面上 比如我在文章里贴一个链接 https://www.steamedfish.top/ ，爬虫爬到这篇文章时就会顺着链接访问我的网站。不过这种方法 见效较慢，因为爬虫不是实时巡逻。

  2. 给搜索引擎提供站点地图（Sitemap）

    站点地图可以告诉爬虫你有哪些页面、哪些更新了，从而让搜索引擎按图索骥，定期抓取。

  3. 在各大搜索引擎上手动提交 URL

    这种方式最快，提交后正常情况下几乎可以立刻进入收录流程，非常适合发布新内容或新站点起步阶段。

提交网站地图（sitemap）#

站点地图一般叫做sitemap.xml, 每当我们的网站有更新站点地图就会更新，谷歌会通过定期的去爬取站点地图。

关于如何生成搜索引擎需要的站点地图

可以查看这篇博客：《如何使用Jekyll生成网站地图（sitemap.xml）的插件jekyll-sitemap》

提示

注意：如果你是用 Hugo 部署的网站，会自动生成 sitemap.xml 文件（伟大，无需多言），通常放在网站根目录下。

以我的博客为例，点击“提交网站地图”，输入 https://www.steamedfish.top/sitemap.xml ，然后提交。。

提交过后后台会花一段时间分析站点，然后状态会显示"成功"，这时你的网站就已经正式录入到 Bing了。
