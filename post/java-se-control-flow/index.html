前言#

在正式开始写代码之前，我们先聊一个非常重要的概念——程序的结构（Program Structures）。
很多新手一上来就盯着语法，看得眼花缭乱，但其实理解程序的“骨架”比死记硬背语法更重要。

Think of it as building a house
你可以买很贵的材料，但没有合理的框架，房子也盖不稳。

程序的基本结构主要有三种：顺序结构（Sequential Structure）、分支结构（Branching Structure）、循环结构（Looping Structure）。

掌握了这三种结构就能实现程序的流程控制
流程控制就是告诉计算机：

什么时候做，做多少次，做完怎么办”

掌握了流程控制，才能写出有逻辑、能做决策和重复操作的程序。

程序结构概览（Program Structures Overview）#

在计算机的世界里，所有的程序无论多复杂，其实都可以由这三种基本结构组合而成：

  1. 顺序结构（Sequential Structure）：语句从上到下依次执行。
  2. 分支结构（Branching Structure）：根据条件执行不同的路径。
  3. 循环结构（Looping Structure）：让某些语句重复执行。

这三种结构看似简单，却是所有算法和程序设计的基础。你可以把它们看作是搭建程序逻辑的“三块积木”，复杂的程序都是在这三种结构的基础上堆叠、嵌套而成的。

顺序结构（Sequential Structure）#

顺序结构是程序中最简单、也是最基础的控制方式。
它遵循“从上往下、依次执行”的原则，每行代码都会被依次执行一次。

来看一个简单的例子：

int a = 5;
int b = 10;
int c = a + b;
System.out.println(c);

程序会先执行第一行，再执行第二行，然后是第三行，最后输出结果 15。

再看这个例子：

int a = 5;
int b = 10;
System.out.println(c);
int c = a + b;

如果我们交换语句的顺序，比如在给 c 赋值之前就打印 c，程序就会报错，因为 c 还不存在。

信息

💬 小知识：输入与输出（Input / Output）

在java中，我们通常用以下两种方式与控制台交互：

Scanner sc = new Scanner(System.in); // 创建输入对象，用于接收键盘输入
System.out.println("请输入一个数字："); // 输出提示信息
int a = sc.nextInt(); // 从控制台读取一个整数
System.out.println("你输入的是：" + a);

System.out.println() 用于输出信息到屏幕；
Scanner 则用来接收用户输入。

顺序结构实例#

接下来我们就用上刚学到的输入与输出来写一个经典的顺序结构程序

求两个数平均值:

import java.util.Scanner;                       //导包，后面学

public class Average {                          //定义类，后面学
   public static void main(String[] args) {    //程序入口
       Scanner sc = new Scanner(System.in);    //创建一个扫描器对象用来接受键盘输入
       System.out.print("请输入两个整数：");   //打印提示信息
       int a = sc.nextInt();                   //从键盘读取一个整数赋值给a
       int b = sc.nextInt();                   //读取一个整数赋值给b
       double avg = (a + b) / 2.0;             //定义一个浮点数(小数)avg，把a+b/2的值赋给它
       System.out.println("平均值是：" + avg); //打印avg
   }
}

分支结构(Branching Structure)#

有时候程序需要根据条件选择不同的路径执行，比如“如果成绩大于60就及格，否则不及格”。
这时候我们就要用到分支结构。

1. if语句#

if语句的语法格式#

  • 格式1：单分支结构

    当条件表达式为true，则执行语句体

    if(条件表达式){
    语句体;
}

  • 格式2：双分支结构

    当条件表达式为true，则执行语句体1，否则执行语句体2

    if(条件表达式){
    语句体1;
}else{
    语句体2;
}

  • 格式3：多分支结构

    当条件表达式1为true，则执行语句体1
    当条件表达式2为true，则执行语句体2
    当条件表达式3为true，则执行语句体3
    …
    当以上条件都为false，则执行语句体n+1

    if(条件表达式1){
    语句体1;
}else if(条件表达式2){
    语句体2;
}else if(条件表达式3){
    语句体3;
}
...
else{
    语句体n+1;
}

提示

if语句可以与其他语句进行嵌套
如：

if(条件表达式1){
    if(条件表达式2){
        语句体2;
    }else{
        语句体3;
    }
}else{
    语句体1;
}

当条件表达式1为true，则进入嵌套判断条件表达式2
当条件表达式2为true，则执行语句体2,否则执行语句体3

当条件表达式1为false，则执行语句体1

2. switch语句#

上面的if...else、if...else 与if...else...if都是多分支结构

switch语句也是多分支结构的一种，当一个变量可能有多种确定取值时，switch 语句比 if...else 更直观和简洁。 比如菜单选择、星期判断、等级判断等场景，就非常适合用 switch。

switch语句的语法格式#

switch(表达式){
    case 常量1:
        语句体1;
        break;
    case 常量2:
        语句体2;
        break;
    ...
    default:
        语句体n+1;
        break;
}

switch语句的执行流程：

  1. 先计算表达式的值
  2. 与每个case的常量进行比较
  3. 如果匹配成功则执行对应的语句体
  4. 若遇到break，则跳出整个switch语句
  5. 如果所有case都不匹配，则执行default

switch语句的"穿透"问题#

上面我们说到：

若遇到break，则跳出整个switch语句

如果没有遇到break呢? 我们以下面这段程序举例：

int day = 3;
switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    case 3:
        System.out.println("星期三");
        //没有break
    case 4:
        System.out.println("星期四");
        //没有break
    default:
        System.out.println("未知日期");
        break;
}

上面这段程序的输出为：

星期三
星期四
未知日期

我们对它的匹配过程进行分析：

case 1: 不匹配，因为 day 是 3。
case 2: 不匹配，因为 day 仍然是 3。
case 3: 匹配成功，执行 System.out.println("星期三");，输出 星期三。
注意：由于没有 break 语句，控制流将继续执行下一个 case。
case 4: 也会被执行，输出 星期四。
由于没有匹配到 case 5，控制流继续到 default:，输出 未知日期。

提示

每个 case 后面通常需要 break 语句，以防止遇到switch语句的穿透问题。

循环结构(Looping Structure)#

在很多程序中，我们会遇到重复执行同一段代码的情况。 比如：

  • 打印 1 到 10 的数字

  • 求数组中所有元素的总和

  • 让用户不断输入直到满足某个条件

如果每次都手动写重复代码，既麻烦又容易出错，这时就需要循环结构（Looping Structure）。

1. while循环#

while 循环先判断条件，如果条件为 true，就执行循环体；循环体执行完，再判断条件，如此重复，直到条件为 false。

while循环的语法格式#

初始化语句;
while(循环条件){
    循环体语句(被重复执行的代码)
    迭代语句;
}

while循环的示例#

打印 1 - 5

int i = 1;
while (i <= 5) {
    System.out.println("i = " + i);
    i++; // 不加 i++ 会无限循环
}

输出结果：

i = 1   
i = 2   
i = 3   
i = 4   
i = 5

2. do…while循环#

do...while 循环和 while 类似，但它先执行一次循环体，再判断条件。
即使条件一开始为 false，循环体也会至少执行一次。

do…while循环的语法格式#

初始化语句
do{
    循环体语句;
    迭代语句;
}while(循环条件);

do…while循环的示例#

int i = 6;
do {    //先执行一遍do{}循环体
    System.out.println("i = " + i);     //输出i
    i++;                                //i自增1
} while (i <= 5);   //i=7，判断i是否<=5，结果为false，结束循环

输出结果：

i = 6

3. for循环#

for 循环是最常用的循环形式，尤其在已知循环次数时。

for循环的的语法格式#

for (初始化 ; 条件表达式 ; 步进(迭代语句)){
    循环体;
}

for循环的示例#

打印 1 - 5

for (int i = 1; i <= 5; i++) {
    System.out.println("i = " + i);
}

输出结果：

i = 1   
i = 2   
i = 3   
i = 4   
i = 5

关键字break和continue#

1. break#

break：直接结束循环

for (int i = 1; i <= 5; i++) {
    if (i == 3) break;
    System.out.println(i);
}

输出结果：

1   
2  //在i=3时，break被执行，跳出当前循环

2. continue#

continue：跳过本次循环

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);
}

输出结果：

1   
2  
4  //在i=3时，continue被执行，跳过本次循环
5

break和continue的区别#

关键字 作用 是否结束循环 典型用途
break 直接跳出循环 ✅ 是 提前退出循环
continue 跳过当前循环 ❌ 否 跳过不需要处理的情况

作用域(Scope)#

作用域是指变量能够被访问到的范围。

理解它非常重要，因为变量只在它被定义的代码块 { } 内有效。

类型 定义位置 作用范围 生命周期
成员变量 类中 整个类 对象存在时
局部变量 方法/语句块中 当前 {} 内 执行完代码块后销毁

局部变量的作用域#

  • 局部变量：定义在方法或代码块中，只能在当前块内使用

示例1：

if (true) {
   int a = 10;
   System.out.println(a); // ✅ 可以访问
}
// System.out.println(a); // ❌ 报错：a 无法访问

a 变量定义在 if 的大括号{}里，所以离开这个大括号后，它就“消失”了。

示例2：

for (int i = 0; i < 3; i++) {
   System.out.println(i);
}
// System.out.println(i); // ❌ 报错：i 在这里无效

变量 i 只在 for 循环的 {}中存在。

成员变量的作用域#

  • 成员变量 又称 全局变量 、 类变量 ：定义在类中，方法外，可以被整个类的所有方法访问 示例：
public class Demo {
   int x = 10; // 成员变量（全局）

   public void test() {   //类中的一个方法，方法名为 test
       int y = 5; // 局部变量
       System.out.println(x + y); // ✅ 都能访问
   }
}
