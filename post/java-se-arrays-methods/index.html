前言#

上一章我们学会了流程控制——让程序知道“什么时候做”“做多少次”。
但如果要让程序处理大量数据，比如学生成绩、商品价格、坐标点列表，你会发现：
光靠一个个变量，代码又臭又长、又难维护。
比如计算三个学生的平均分，你可能会写出这样的代码：

int a = 90, b = 80, c = 70;
int sum = a + b + c;
double avg = sum / 3.0;
System.out.println("平均分：" + avg);

要是有十个学生呢？一百个呢？
这时我们就需要一个能一次性装下多个数据的“容器”——数组(Array)。
以及一个能封装、复用代码逻辑的工具——方法(Method)。

数组(Array)#

1.数组的概念#

数组是存放同一类型数据的有序集合。

  • 同一类型：数组里只能放整数、浮点数、字符串等同一种类型。
  • 有序集合：每个元素都有自己的位置（下标）。

可以把它想象成一个带编号的格子柜，每个格子放一个数据，每个格子都有编号（下标）。

📦 “数组就是一组有序排列的数据箱子。”

2.为什么要用数组？#

假设你要存10个成绩，不用数组就得写10个变量：

int s1, s2, s3, s4, s5, s6, s7, s8, s9, s10;

光是声明就够让人崩溃了，更别提算平均数。
数组可以让你用一个变量名表示一组相同类型的数据：

int[] score = new int[10];

这行代码就创建了一个能装 10 个整数的“容器”。

3.数组的声明与创建(Declaration and Creation)#

数据类型[] 数组名;             // 声明数组
数组名 = new 数据类型[长度];    // 创建数组

示例：

int[] scores;        // 声明一个整数数组
scores = new int[5]; // 创建一个长度为5的数组

或者合并写成一行：

int[] scores = new int[5];

信息

提示：数组长度一旦创建就固定，不能动态改变。 如果要动态扩展，得使用“容器类”（比如 ArrayList），后面我们会讲。

4.数组的初始化(Initialization)#

数组创建出来之后，还需要“放东西进去”——这就是初始化。

Java中有三种常见方式：

静态初始化#

适用于：已知每个元素时使用

  • 格式 
    数据类型[] 数组名 = {元素1,元素2,元素3...};
//必须在一个语句中完成，不能分开两个语句写
  • 示例
int[] scores = {90, 80, 70, 85, 95};

动态初始化#

适用于：先确定长度，稍后再赋值。

  • 格式： 
    数据类型[] 数组名 = new 数据类型[长度];
数组名[下标] = 值;
  • 示例
int[] scores = new int[5];//创建一个长度为5的整形数组，五个元素的下标为 0, 1, 2, 3, 4
scores[0] = 90;             //下标为0的元素值为90
scores[1] = 80;             //下标为1的元素值为80
scores[2] = 70;             //下标为2的元素值为70
scores[3] = 85;             //下标为3的元素值为85
scores[4] = 95;             //下标为4的元素值为95

信息

注意： 数组下标从0开始，最后一个下标是 长度- 1

访问越界会导致 报错Array(数组)Index(索引)OutOfBounds(越界)Exception(异常)

混合写法#

其实静态初始化和动态初始化也能结合

int[] scores = new int[]{90, 80, 70, 85, 95};

不过这种方式在现代Java中较少使用。

5.数组元素的默认值(Hybrid Initialization)#

当你用 new 创建一个数组时，Java 会自动为每个元素赋初始值：

数据类型 默认值
byte, short, int, long 0
float, double 0.0
char '\u0000'（空字符）
boolean false
引用类型（对象） null

示例：

int[] nums = new int[3];     //初始化一个整形数组
//不手动为元素赋值，整形数组的元素默认值为0
System.out.println(nums[0]); // 输出 0

6.数组的遍历和访问(Access and Traverse)#

前面提过，数组中的每个元素都有自己的编号，叫下标（index），从 0 开始。

例如：

int[] nums = {10, 20, 30, 40, 50};

它的下标与值的关系是这样的：

下标 值
0 10
1 20
2 30
3 40
4 50

使用下标访问：#

要访问第三个元素

System.out.println(nums[2]); // 输出30
// 输出下标为2的元素

如果我们要把第一个元素从10改成99呢？

nums[1] = 99;  // 把第二个元素改为99

使用for循环访问：#

数组遍历是指依次访问每个元素。最常见的是 for 循环：

int[] nums = {10, 20, 30, 40, 50};
for (int i = 0; i < nums.length; i++) {
    System.out.println(nums[i]);
}

输出结果：

10
20
30
40
50

信息

⚠️ 注意： length 是属性（没有括号），而不是方法。 不要写成 scores.length()。

使用增强for循环(for-each)#

如果只是读取数组内容，不修改，可以使用更简洁的写法：

for (int s : nums) {
    System.out.println(s);
}

输出结果：

10
20
30
40
50

这表示“依次把数组 nums 的每个元素赋值给变量 n”。 但是要注意：

增强型 for 不能修改数组的内容，因为 n 是副本。

信息

8.二维数组(2D Arrays)#

数组中还能存数组，这就是“二维数组”：

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

访问方式：

System.out.println(matrix[0][1]); // 输出第1行第2列的元素 2

二维数组常用于存储表格、棋盘等数据结构。

9.数组的应用实例#

查找最大值#

int[] arr = {3, 5, 7, 2, 9};
int max = arr[0];
for (int i = 1; i < arr.length; i++) >{
   if (arr[i] > max) {
       max = arr[i];
   }
}
System.out.println("最大值：" + max);

数组反转#

上强度了喔

int[] arr = {1, 2, 3, 4, 5};
for (int i = 0; i < arr.length / 2; i++) {
   int temp = arr[i];
   arr[i] = arr[arr.length - 1 - i];
   arr[arr.length - 1 - i] = temp;
}
System.out.println(Arrays.toString>(arr)); // [5, 4, 3, 2, 1]

方法(Method)#

数组让我们高效管理数据，
方法则让我们高效复用逻辑。

1.方法的概念#

方法是可以被重复调用的代码块，用来完成特定功能。
就像现实中“函数”或“工具”：
写一次，用多次。

例如，计算两个数的和：

int sum = a + b;

如果你需要多次计算不同的加法，就可以写成方法：

public static int add(int a, int b) {
   return a + b;
}

然后在 main()(程序入口/main函数) 里直接调用：

int result = add(3, 5);
System.out.println(result); // 输出8

2.方法的定义格式#

修饰符 返回类型 方法名(参数列表) {
    // 方法体
    return 返回值;
}

  • 修饰符：一般先写 public static

  • 返回类型：方法返回的数据类型（无返回值写 void）

  • 方法名：遵循小驼峰命名，如 getSum

  • 参数列表：要传入的变量（可有多个）

  • return：返回一个结果

3.方法的调用#

在同一个类中:

public static void main(String[] args) {//主程序
    sayHello(); // 调用方法
}                                       //程序结束

public static void sayHello() {
    System.out.println("Hello Java!");
}

4.有参和无参方法#

无参方法#

public static void printLine() {
    System.out.println("===========");
}

调用:

printLine();

有参方法#

public static void printSum(int a, int b) {
    System.out.println("和为：" + (a + b));
}

调用：

printSum(10, 20);

输出:

===========

5.带返回值的方法#

public static int getMax(int a, int b) {
    if (a > b) return a;
    else return b;
}

调用：

int max = getMax(3, 8);
System.out.println("最大值：" + max);

输出:

8

6.方法和数组结合#

方法也能处理数组，非常常见

只要把数组当作参数传入一个可以接受数组的方法即可：

public static int getSum(int[] arr) {
    int sum = 0;
    for (int n : arr) {
        sum += n;
    }
    return sum;
}

调用

int[] nums = {1, 2, 3, 4, 5};
System.out.println(getSum(nums)); // 输出15

输出:

15

可变参数和方法重载#

1.可变参数(Varargs)#

在前面的例子中，我们定义的方法参数个数是固定的

例如：

public static int add(int a, int b) {
    return a + b;
}

但如果我们想计算 任意多个数的和，
比如：

add(1, 2);
add(1, 2, 3);
add(5, 10, 15, 20);

就没法写一个固定参数的方法了。

这时我们就可以使用 可变参数（Variable Arguments）。

2.可变参数的定义#

修饰符 返回类型 方法名(参数类型... 参数名) { }

重点是参数类型后的 …

实例：

public static int sum(int... nums) {
   int sum = 0;
   for (int n : nums) {
       sum += n;
   }
   return sum;
}

调用：

System.out.println(sum(1, 2));            // 输出3
System.out.println(sum(1, 2, 3));         // 输出6
System.out.println(sum(5, 10, 15, 20));   // 输出50

3.可变参数的本质#

可变参数本质上就是一个数组。 上面的 int... nums 在方法内部等价于：

int[] nums;

也就是说：

sum(1, 2, 3);

会被 Java 自动转换成：

sum(new int[]{1, 2, 3});

信息

可变参数在一个方法中只能有一个，

并且必须是参数列表的最后一个参数。

4.方法重载#

方法重载 是指在同一个类中，允许多个方法 名字相同，但 参数不同（数量、类型或顺序不同）。

示例：

public static int add(int a, int b) >{
   return a + b;
}

public static double add(double a, double b) {
    return a + b;
}

public static int add(int a, int b, int c) {
    return a + b + c;
}

调用：

System.out.println(add(1, 2));         // 调用第一个方法
System.out.println(add(1.5, 2.5));     // 调用第二个方法
System.out.println(add(1, 2, 3));      // 调用第三个方法

输出：

3
4.0
6

5.方法重载的匹配规则#

当 Java 编译器看到多个同名方法时，会根据参数类型和数量自动匹配最合适的版本。

优先级顺序为：

  1. 精确匹配（参数类型完全一致）

  2. 自动类型提升（如 int → double）

  3. 可变参数匹配（最后兜底） 例如：

public static void show(int a) {
    System.out.println("int");
}
public static void show(double a) {
    System.out.println("double");
}
public static void show(int... a) {
    System.out.println("varargs");
}

show(10);       // 输出 int（精确匹配）
show(10.5);     // 输出 double
show();         // 输出 varargs（没有参数时调用）

6.方法重载的优点#

  • 语义统一：相同功能用一个名字即可，逻辑更清晰。
  • 灵活易用：可处理不同类型或数量的输入。
  • 提高代码可读性

综合应用实例#

计算任意学生的成绩的平均值

public class ScoreTool {
    public static double avg(int... scores) {
        int sum = 0;
        for (int s : scores) sum += s;
        return (double) sum / scores.length;
    }

    public static double avg(double... scores) {
        double sum = 0;
        for (double s : scores) sum += s;
        return sum / scores.length;
    }

    public static void main(String[] args) {
        System.out.println(avg(80, 90, 100));         // int 版本
        System.out.println(avg(88.5, 92.0, 76.5));    // double 版本
    }
}

输出：

90.0
85.66666666666667
