String 类#

1. String的概念与不可变性#

String 是 Java 中最常用的类，用来表示一串字符，比如 “Hello”、“Java”。
但有一个关键点必须记住：

String 对象是不可变的（immutable） 也就是说，一旦创建，就不能修改。

String s = "Hello";
s = s + " Java";

这里并不是修改了 s，而是创建了一个新的字符串 “Hello Java”。 原来的 “Hello” 被丢弃。

这就是为什么字符串拼接频繁时性能会下降。

2.创建方式#

String s1 = "Hello";           // 直接赋值（推荐）
String s2 = new String("Hi");  // 通过构造方法

第一种方式会把字符串放入“字符串常量池”，节省内存。 第二种每次都会创建新对象。

3.String的常用方法#

.length()获取长度#

String str = "hello";
int len = str.length(); // 5

注意：返回的是字符数，不是字节数，中文也算一个字符。

.charAt()取指定下标字符#

char c = "java".charAt(1); // 'a'

注意：索引从 0 开始，越界会抛出 StringIndexOutOfBoundsException。

.equals()和.equalsIgnoreCase()内容比较#

boolean b1 = "abc".equals("abc");           // true
boolean b2 = "abc".equalsIgnoreCase("ABC"); // true

equals 区分大小写，equalsIgnoreCase 忽略大小写。

.contains()是否包含子串#

boolean b = "hello".contains("he"); // true

.startsWith()是否以指定前缀/.endsWith()后缀开头/结尾#

"hello".startsWith("he"); // true
"hello".endsWith("lo");   // true

.indexOf()和.lastIndexOf()查找子串位置#

int first = "hello".indexOf("l");       // 2
int last  = "hello".lastIndexOf("l");   // 3

如果未找到返回 -1。

.substring()截取子串#

String s1 = "hello".substring(2);       // "llo"
String s2 = "hello".substring(1, 4);    // "ell"

注意：substring(begin, end) 包含 begin，不包含 end。

.toLowerCase()和.toUpperCase()转大小写#

"JAVA".toLowerCase(); // "java"
"java".toUpperCase(); // "JAVA"

.trim()去除首尾空格#

String s = "  abc  ".trim(); // "abc"

中间的空格不会被去掉。

.replace()替换字符或子串#

String s1 = "hello".replace('l', 'x');          // "hexxo"
String s2 = "hello world".replace("world", "Java"); // "hello Java"

注意：replace 会返回新字符串，原字符串不变。

.split()分割字符串#

String s = "apple,banana,orange";
String[] arr = s.split(","); // ["apple", "banana", "orange"]

参数是正则表达式，可以用 \s+ 按空格分割等。

.isEmpty()判断是否为空#

String s1 = "";
boolean b1 = s1.isEmpty();  // true
String s2 = null;
boolean b2 = (s2 == null || s2.isEmpty()); // true

空串指长度为 0，不包括 null。isEmpty() 不能直接调用在 null 上，会抛出 NullPointerException。

.concat()拼接字符串：两两拼接#

String s1 = "Hello";
String s2 = "World";
String s3 = s1.concat(s2); // "HelloWorld"

注意：与 + 类似，仍会创建新对象，频繁拼接效率低。

.join()拼接字符串：多元素#

String s = String.join(", ", "apple", "banana", "orange"); 
// "apple, banana, orange"
