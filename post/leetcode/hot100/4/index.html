【HOT100-hard】4.寻找两个正序数组的中位数

2025-12-30 · 3 分钟 |  编辑此文章
目录
  • 题目

题目#

给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

算法的时间复杂度应该为 O(log (m+n)) 。

示例 1：

输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2

示例 2：

输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5

题解#

用到了二分查找，首先我们要保证nums1是两个数组中较小的一个。然后我们来看
数组nums1长度为m
数组nums2长度为n

我们暂时不考虑两个数组的情况，先来只有nums1数组的情况下：
我们想象有一根 切割线，切割线左边的数量为i，

  • 当数组为奇数时，i = (m + 1) / 2 
    [ x  x  x | x  x ]
        ↑ 中位数
    • i = (5 + 1) / 2 = 3
    • 中位数 = A[i-1]
  • 当数组为偶数时，也满足i = (m + 1) / 2 
    [ x  x  x | x  x  x ]
        ↑   ↑
       中间两个
    • i = (6 + 1) / 2 = 3
    • 中位数 = (A[i-1] + A[i]) / 2

有两个数组时，
A和B的总长度是m+n，切割线同时切割nums1和nums2

  • nums1中切割线左边的数量为i
  • nums2中切割线左边的数量为j
  • i + j =左半部分总数量= (m + n + 1) / 2
nums1 = [1, 3]      (m = 2)
nums2 = [2, 4, 5]   (n = 3)

总长度 = 5
左半部分应该有 (5 + 1) / 2 = 3 个
在满足左半部分的数量 =(m + n + 1) / 2的情况下，调整i的位置直到切割线的位置正确。就能根据切割线找到中位数。

设nums1切割线左边的数字为left1，右边的数字为right1
设nums2切割线左边的数字为left2，右边的数字为right2
当切割线正确时，左边的数字应该小于右边的数字。又因为数组是有序的
那么只要满足left1<=right2且left2<=right1，切割线就是正确的。

同时我们还要注意边界处理（其中一个数组全都小于或者大于另一个数组）：
例如nums1全都<= nums2，此时i = m，right1应该取+∞，在java中应该是Integer.MAX_VALUE。它只是用来占位的，来让left2<=rifht2恒成立。

在调整i的过程，就是一个在 0到m的区间搜索i的过程
这个搜索区间的左边界称为left，右边界称为right
这个时候我们就要用到二分查找了。
一开始我们在nums1的中点位置（(left + right) / 2）取i
假如不满足left1 <= right2，说明什么？
说明left1 > right2 –> left1太大 –> 要知道这是个有序数组，i右边的数都大于i（中点右边） –> i和i右边的数字都可以被排除出搜索区间，右边界可以直接改变，也就是 right = nums1[i - 1]，通过这样一次排除一半的搜索区间，我们就可以很快速的找出正确的i和切割线

class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        if(nums1.length>nums2.length){
            return findMedianSortedArrays(nums2,nums1);
            // 调换位置
        }
        int m = nums1.length;
        int n = nums2.length;
        int left = 0,right = m;
        while(left <= right){
            // 初始化 i
            // int i = (left + right) / 2;
            // 防止溢出写法
            int i = left + (right -left) /2;
            // i + j 满足 i+j = (m+n+1)/2
            int j = (m + n + 1) / 2 - i;

            // 边界处理（难点）
            int left1 = (i == 0) ? Integer.MIN_VALUE : nums1[i];
            int right1 = (i == m) ? Integer.MAX_VALUE : nums1[i];
            int left2 = (j == 0) ? Integer.MIN_VALUE : nums2[j];
            int right2 = (j == n) ? Integer.MAX_VALUE : nums2[j];

            if(left1 <= right2 && left2 <= right1){
                // 正确切割线
                if((m + n) % 2 == 1){ //奇数
                    return (double) (Math.Max(left1, left2));
                }else{  //偶数 （难理解）
                    return (double) (Math.Max(left1, left2) + Math.Min(right1, right2) / 2)
                }
            }else if(left1 > right2){   //二分查找思想（难点）
                right = i - 1;
            }else{
                left = i + 1;
            }
        }
        return 0.0;
    }
}
