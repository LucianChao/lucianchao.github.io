Java I/O 与文件操作#

1.I/O 概述#

Java 提供了丰富的 输入输出（I/O） 机制，用于处理数据的读写，包括文件、控制台、网络等。

  • 输入流（InputStream / Reader）：从外部读取数据到程序中。

  • 输出流（OutputStream / Writer）：将数据写出到外部。

  • 字节流 vs 字符流：

    • 字节流：InputStream / OutputStream，按字节处理，适合所有类型文件（文本、图片、视频）。

    • 字符流：Reader / Writer，按字符处理，适合文本文件，自动处理字符编码。

  • 流操作基本步骤：

    1. 打开流（new 对象）

    2. 读写数据（read()/write()）

    3. 关闭流（close()），最好使用 try-with-resources 自动关闭

注意：不要忘记关闭流，否则可能导致内存泄漏或文件锁定。

2.字节流#

字节流用于读取或写入二进制数据。

FileInputStream / FileOutputStream#

import java.io.*;

public class ByteStreamDemo {
    public static void main(String[] args) {
        // 写入文件
        try (FileOutputStream fos = new FileOutputStream("test.dat")) {
            String data = "Hello Java I/O";
            fos.write(data.getBytes()); // 字符串转字节数组
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 读取文件
        try (FileInputStream fis = new FileInputStream("test.dat")) {
            byte[] buffer = new byte[1024];
            int len = fis.read(buffer);
            String content = new String(buffer, 0, len);
            System.out.println(content);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

注意：

  • read() 方法返回读取的字节数，若返回 -1 表示文件末尾。
  • 每次读取字节会覆盖 buffer 的内容，要根据返回长度构造字符串。

3.字符流与缓冲流#

字符流专门处理文本数据，按字符读取，自动处理编码问题。

FileReader / FileWriter#

import java.io.*;

public class CharStreamDemo {
    public static void main(String[] args) {
        // 写入文本文件
        try (FileWriter fw = new FileWriter("test.txt")) {
            fw.write("Hello Java Character Stream\n");
            fw.write("第二行内容");
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 读取文本文件
        try (FileReader fr = new FileReader("test.txt")) {
            char[] buffer = new char[1024];
            int len = fr.read(buffer);
            String content = new String(buffer, 0, len);
            System.out.println(content);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

注意：

  • FileReader / FileWriter 适合文本文件，自动处理字符编码。
  • read(char[] buffer) 方法返回实际读取的字符数，遇到文件末尾返回 -1。

缓冲流 BufferedReader / BufferedWriter#

缓冲流通过内部缓冲区提高读写效率，同时提供便捷的方法，如 readLine()。

import java.io.*;

public class BufferedStreamDemo {
    public static void main(String[] args) {
        // 写入
        try (BufferedWriter bw = new BufferedWriter(new FileWriter("buffer.txt"))) {
            bw.write("第一行内容");
            bw.newLine(); // 换行
            bw.write("第二行内容");
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 读取
        try (BufferedReader br = new BufferedReader(new FileReader("buffer.txt"))) {
            String line;
            while ((line = br.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

注意：

  • BufferedReader.readLine() 不包含换行符。
  • 缓冲流适合大文件读取，避免频繁 I/O 调用，提高性能。

4. PrintWriter#

PrintWriter 可以方便地写文本，同时支持自动刷新。

import java.io.*;

public class PrintWriterDemo {
    public static void main(String[] args) {
        try (PrintWriter pw = new PrintWriter(new FileWriter("print.txt"))) {
            pw.println("第一行");
            pw.println("第二行");
            pw.printf("Hello %s, 年龄 %d\n", "Alice", 20);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

注意：

  • println() 自动添加换行符。
  • 可以结合 printf() 格式化输出。
  • 构造器可以传入 autoFlush = true 来实现自动刷新。

5. RandomAccessFile#

RandomAccessFile 支持随机访问文件，可以读写任意位置。

import java.io.*;

public class RandomAccessFileDemo {
    public static void main(String[] args) {
        try (RandomAccessFile raf = new RandomAccessFile("random.txt", "rw")) {
            raf.writeUTF("Hello Java");
            raf.seek(0); // 回到文件开头
            String s = raf.readUTF();
            System.out.println(s);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

注意：

  • 模式 “rw” 支持读写。
  • seek(long pos) 设置文件指针位置。
  • 适合对文件中任意位置数据修改。

6.文件操作 API（java.io.File）#

File 代表文件或目录，但不负责内容读写，仅用于管理。

import java.io.File;

public class FileDemo {
    public static void main(String[] args) {
        File f = new File("demo.txt");
        if (!f.exists()) {
            try {
                f.createNewFile();
                System.out.println("文件创建成功");
            } catch (Exception e) {
                e.printStackTrace();
            }
        }

        System.out.println("文件名: " + f.getName());
        System.out.println("绝对路径: " + f.getAbsolutePath());
        System.out.println("是否存在: " + f.exists());
        System.out.println("是否是文件: " + f.isFile());
        System.out.println("是否是目录: " + f.isDirectory());
    }
}

注意：

  • File 也可以表示目录，用于创建、删除、判断等操作。
  • 删除文件或目录用 delete()，但删除目录要求目录为空。
  • 获取子文件可用 list() 或 listFiles()。

7. NIO文件操作（java.nio.file）#

Java 7 引入了 NIO（New IO），提供更现代化的文件操作 API。

import java.nio.file.*;
import java.io.IOException;
import java.util.List;

public class NioFileDemo {
    public static void main(String[] args) {
        Path path = Paths.get("nio.txt");

        // 写入文件（覆盖）
        try {
            Files.write(path, "Hello NIO".getBytes());
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 读取文件
        try {
            List<String> lines = Files.readAllLines(path);
            for (String line : lines) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 复制文件
        try {
            Files.copy(path, Paths.get("nio_copy.txt"), StandardCopyOption.REPLACE_EXISTING);
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 删除文件
        try {
            Files.deleteIfExists(Paths.get("nio_copy.txt"));
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

注意：

  • Paths.get() 用于创建路径对象。
  • Files 提供了静态方法读写文件、复制、移动、删除等。
  • NIO 支持更高效的大文件操作和异步 IO（高级用法可选）。

8.字符集与编码#

Java 中字符编码相关类主要是 Charset，确保读写中文或其他特殊字符时不乱码。

import java.nio.file.*;
import java.nio.charset.*;
import java.io.IOException;
import java.util.List;

public class CharsetDemo {
    public static void main(String[] args) {
        Path path = Paths.get("charset.txt");

        // 指定 UTF-8 写入
        try {
            Files.write(path, "你好，Java NIO".getBytes(StandardCharsets.UTF_8));
        } catch (IOException e) {
            e.printStackTrace();
        }

        // 指定 UTF-8 读取
        try {
            List<String> lines = Files.readAllLines(path, StandardCharsets.UTF_8);
            for (String line : lines) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

注意：

  • 默认编码可能随系统不同而变化（Windows 默认 GBK，Linux 默认 UTF-8）。
  • 推荐使用 StandardCharsets.UTF_8 显式指定编码，避免中文乱码。
