String 类#

1. String的概念与不可变性#

String 是 Java 中最常用的类，用来表示一串字符，比如 “Hello”、“Java”。
但有一个关键点必须记住：

String 对象是不可变的（immutable） 也就是说，一旦创建，就不能修改。

String s = "Hello";
s = s + " Java";

这里并不是修改了 s，而是创建了一个新的字符串 “Hello Java”。 原来的 “Hello” 被丢弃。

这就是为什么字符串拼接频繁时性能会下降。

2.创建方式#

String s1 = "Hello";           // 直接赋值（推荐）
String s2 = new String("Hi");  // 通过构造方法

第一种方式会把字符串放入“字符串常量池”，节省内存。 第二种每次都会创建新对象。

3.String的常用方法#

.length()获取长度#

String str = "hello";
int len = str.length(); // 5

注意：返回的是字符数，不是字节数，中文也算一个字符。

.charAt()取指定下标字符#

char c = "java".charAt(1); // 'a'

注意：索引从 0 开始，越界会抛出 StringIndexOutOfBoundsException。

.equals()和.equalsIgnoreCase()内容比较#

boolean b1 = "abc".equals("abc");           // true
boolean b2 = "abc".equalsIgnoreCase("ABC"); // true

equals 区分大小写，equalsIgnoreCase 忽略大小写。

.contains()是否包含子串#

boolean b = "hello".contains("he"); // true

.startsWith()是否以指定前缀/.endsWith()后缀开头/结尾#

"hello".startsWith("he"); // true
"hello".endsWith("lo");   // true

.indexOf()和.lastIndexOf()查找子串位置#

int first = "hello".indexOf("l");       // 2
int last  = "hello".lastIndexOf("l");   // 3

如果未找到返回 -1。

.substring()截取子串#

String s1 = "hello".substring(2);       // "llo"
String s2 = "hello".substring(1, 4);    // "ell"

注意：substring(begin, end) 包含 begin，不包含 end。

.toLowerCase()和.toUpperCase()转大小写#

"JAVA".toLowerCase(); // "java"
"java".toUpperCase(); // "JAVA"

.trim()去除首尾空格#

String s = "  abc  ".trim(); // "abc"

中间的空格不会被去掉。

.replace()替换字符或子串#

String s1 = "hello".replace('l', 'x');          // "hexxo"
String s2 = "hello world".replace("world", "Java"); // "hello Java"

注意：replace 会返回新字符串，原字符串不变。

.split()分割字符串#

String s = "apple,banana,orange";
String[] arr = s.split(","); // ["apple", "banana", "orange"]

参数是正则表达式，可以用 \s+ 按空格分割等。

.isEmpty()判断是否为空#

String s1 = "";
boolean b1 = s1.isEmpty();  // true
String s2 = null;
boolean b2 = (s2 == null || s2.isEmpty()); // true

空串指长度为 0，不包括 null。isEmpty() 不能直接调用在 null 上，会抛出 NullPointerException。

.concat()拼接字符串：两两拼接#

String s1 = "Hello";
String s2 = "World";
String s3 = s1.concat(s2); // "HelloWorld"

注意：与 + 类似，仍会创建新对象，频繁拼接效率低。

.join()拼接字符串：多元素#

String s = String.join(", ", "apple", "banana", "orange"); 
// "apple, banana, orange"

StringBuilder 类#

1.StringBuilder 的概念#

StringBuilder 是可变字符串类，用于频繁修改字符串时性能优于 String。

线程不安全，但单线程效率高。

2.StringBuilder 的常用方法#

public class StringBuilderDemo {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder("Hello");

        // append() 添加
        sb.append(" Java");
        System.out.println(sb); // Hello Java

        // insert() 插入
        sb.insert(6, "World ");
        System.out.println(sb); // Hello World Java

        // delete() 删除指定范围（包前不包后）
        sb.delete(6, 12);
        System.out.println(sb); // Hello Java

        // deleteCharAt() 删除指定下标字符
        sb.deleteCharAt(5);
        System.out.println(sb); // HelloJava

        // replace() 替换指定范围（包前不包后）
        sb.replace(5, 9, " World");
        System.out.println(sb); // Hello World

        // reverse() 反转
        sb.reverse();
        System.out.println(sb); // dlroW olleH

        // charAt() 获取字符
        char c = sb.charAt(0);
        System.out.println(c); // d

        // setCharAt() 修改指定下标字符
        sb.setCharAt(0, 'D');
        System.out.println(sb); // DlroW olleH

        // length() 获取长度
        System.out.println(sb.length()); // 11

        // capacity() 获取容量
        System.out.println(sb.capacity()); // 默认 16+当前长度
    }
}

注意：StringBuilder 修改原对象，不会产生新对象。

StringBuffer 类#

1. StringBuffer 的概念#

StringBuffer 与 StringBuilder 类似，也是可变字符串。

线程安全，适合多线程环境，但效率略低于 StringBuilder。

2. 常用方法#

StringBuffer 的方法几乎和 StringBuilder 一样：

append()、insert()、delete()、deleteCharAt()、replace()、reverse()、charAt()、setCharAt()、length()、capacity()

StringBuffer sb = new StringBuffer("Hello");
sb.append(" Java");
System.out.println(sb); // Hello Java

区别主要是线程安全：方法加了 synchronized，多线程下不会出现数据错乱。

StringJoiner 类#

1. StringJoiner 的概念#

StringJoiner 是 Java 8 新增，用于拼接字符串，支持指定分隔符、前缀和后缀。

适合拼接集合或数组元素，灵活控制格式。

2.常用方法#

import java.util.StringJoiner;

public class StringJoinerDemo {
    public static void main(String[] args) {
        // 使用分隔符
        StringJoiner sj = new StringJoiner(", ");
        sj.add("Apple");
        sj.add("Banana");
        sj.add("Orange");
        System.out.println(sj); // Apple, Banana, Orange

        // 使用前缀和后缀
        StringJoiner sj2 = new StringJoiner(", ", "[", "]");
        sj2.add("Java");
        sj2.add("Python");
        System.out.println(sj2); // [Java, Python]

        // merge() 合并两个 StringJoiner
        StringJoiner sj3 = new StringJoiner(" | ");
        sj3.add("C");
        sj3.add("C++");
        sj2.merge(sj3);
        System.out.println(sj2); // [Java, Python, C | C++]
    }
}

注意：StringJoiner 不可变，add 会修改对象本身。

Arrays类#

Arrays是工具类，用于操作数组，常用方法很多。

import java.util.Arrays;

public class ArraysDemo {
    public static void main(String[] args) {
        int[] arr = {5, 2, 8, 1};

        // sort() 排序
        Arrays.sort(arr);
        System.out.println(Arrays.toString(arr)); // [1, 2, 5, 8]

        // binarySearch() 二分查找，数组必须先排序
        int index = Arrays.binarySearch(arr, 5);
        System.out.println(index); // 2

        // copyOf() 复制数组
        int[] arr2 = Arrays.copyOf(arr, arr.length);
        System.out.println(Arrays.toString(arr2)); // [1, 2, 5, 8]

        // copyOfRange() 复制指定范围
        int[] arr3 = Arrays.copyOfRange(arr, 1, 3);
        System.out.println(Arrays.toString(arr3)); // [2, 5]

        // fill() 填充数组
        int[] arr4 = new int[5];
        Arrays.fill(arr4, 9);
        System.out.println(Arrays.toString(arr4)); // [9, 9, 9, 9, 9]

        // equals() 比较数组内容
        System.out.println(Arrays.equals(arr, arr2)); // true

        // toString() 转字符串
        System.out.println(Arrays.toString(arr)); // [1, 2, 5, 8]

        // deepToString() 多维数组
        int[][] arr5 = {{1,2},{3,4}};
        System.out.println(Arrays.deepToString(arr5)); // [[1, 2], [3, 4]]
    }
}

注意：

  • binarySearch 必须在有序数组上使用，否则结果不正确。
  • 多维数组比较需要 deepEquals 或 deepToString。

Math类#

Math是数学工具类，提供基本数学运算。

public class MathDemo {
    public static void main(String[] args) {
        double x = -3.5, y = 2.7;

        // abs() 绝对值
        System.out.println(Math.abs(x)); // 3.5

        // ceil() 向上取整
        System.out.println(Math.ceil(y)); // 3.0

        // floor() 向下取整
        System.out.println(Math.floor(y)); // 2.0

        // round() 四舍五入
        System.out.println(Math.round(y)); // 3

        // max() / min()
        System.out.println(Math.max(x, y)); // 2.7
        System.out.println(Math.min(x, y)); // -3.5

        // pow() 幂
        System.out.println(Math.pow(2, 3)); // 8.0

        // sqrt() 平方根
        System.out.println(Math.sqrt(16)); // 4.0

        // random() 生成 [0,1) 随机数
        System.out.println(Math.random());
    }
}

注意：

  • Math.random() 返回 double 类型，范围 [0,1)。
  • ceil 和 floor 返回 double，round 返回 long。

时间类(Time)#

Java 8 引入了新的时间 API（java.time 包），推荐使用 LocalDate、LocalTime、LocalDateTime 等类。

import java.time.LocalDate;
import java.time.LocalTime;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

public class TimeDemo {
    public static void main(String[] args) {
        // LocalDate 获取日期
        LocalDate date = LocalDate.now();
        System.out.println(date); // 2025-10-31

        // LocalTime 获取时间
        LocalTime time = LocalTime.now();
        System.out.println(time); // 19:45:23.123

        // LocalDateTime 获取日期时间
        LocalDateTime dateTime = LocalDateTime.now();
        System.out.println(dateTime); // 2025-10-31T19:45:23.123

        // 自定义格式化
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy/MM/dd HH:mm:ss");
        String str = dateTime.format(formatter);
        System.out.println(str); // 2025/10/31 19:45:23

        // 解析字符串为时间
        LocalDateTime dt2 = LocalDateTime.parse("2025/10/31 19:45:23", formatter);
        System.out.println(dt2); // 2025-10-31T19:45:23

        // 日期运算
        LocalDate tomorrow = date.plusDays(1);
        LocalDate yesterday = date.minusDays(1);
        System.out.println(tomorrow);
        System.out.println(yesterday);
    }
}

注意：

  • LocalDate、LocalTime、LocalDateTime 都是不可变对象，方法如 plusDays 返回新对象。
  • DateTimeFormatter 用于格式化和解析，格式必须匹配。

BigDecimal类#

BigDecimal 用于高精度运算，尤其适合金融计算。

import java.math.BigDecimal;
import java.math.RoundingMode;

public class BigDecimalDemo {
    public static void main(String[] args) {
        BigDecimal a = new BigDecimal("10.123");
        BigDecimal b = new BigDecimal("3.12");

        // 加
        BigDecimal sum = a.add(b);
        System.out.println(sum); // 13.243

        // 减
        BigDecimal diff = a.subtract(b);
        System.out.println(diff); // 7.003

        // 乘
        BigDecimal prod = a.multiply(b);
        System.out.println(prod); // 31.58776

        // 除
        BigDecimal div = a.divide(b, 2, RoundingMode.HALF_UP);
        System.out.println(div); // 3.25 （保留 2 位小数，四舍五入）

        // 设置小数位
        BigDecimal scaled = a.setScale(1, RoundingMode.DOWN);
        System.out.println(scaled); // 10.1

        // 比较大小
        int cmp = a.compareTo(b); // 1 表示 a > b
        System.out.println(cmp);
    }
}

注意：

  • 构造 BigDecimal 时最好使用字符串，以避免浮点数精度误差。
  • BigDecimal 对象不可变，运算会生成新对象。
  • 除法必须指定舍入方式，否则可能抛出 ArithmeticException。
