HTML快速入门#

  1. 新建文本文件，后缀名为.html
  2. 编写HTML的基本框架，定义标题 
    <html>
 <head>
     <title>HTML快速入门</title>
     <!-- 网页头部：用来存放给浏览器的信息，
     如：CSS样式 -->
 </head>
 <body>
     <h1>Hello World!</h1>
     <img src="1.jpg">
     <!-- 网页主体：用来存放给用户看的信息，
     如：文字、图片、视频 -->

 </body>
 </html>
  3. 在<body>中填写内容

HTML标签的特点#

  1. html标签不区分大小写，建议小写
  2. html标签的属性值使用单引号/双引号都可以
  3. html语法结构松散，但是建议规范书写

标准HTML文件#

<!-- 声明文档类型 -->
<!DOCTYPE v-html="html">
<html lang="en">

<head>
    <!-- 文档类型 -->
    <meta charset="UTF-8" />
    <!-- 解决浏览器兼容问题 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <!-- 标题  -->
    <title>HTML快速入门</title>
</head>

<body>
    <h1>Hello Html</h1>
    <img src="../1.jpg">
</body>
</html>

数据采集标签#

这些标签用于 接收用户输入，主要是表单元素（Form Elements）。

  • 表单容器：<form>：用于定义整个表单区域。
    • action：定义提交数据的路径
    • method：定义表单提交方式
      • get：请求数据直接 & 拼接在路径?后面（一般用于请求资源），大小有限制
      • post：请求数据通过请求体(可在网页开发人员工具->网络中进行查看)传递（一般用于提交数据），大小无限制
  • 常用输入控件
    • <input>：输入框（可设置 type=“text”、“password”、“checkbox” 等）
      • type：要显示的元素的类型。
        • text：文本
        • password：密码
        • radio：单选按钮
        • checkbox：复选框
        • file：文件上传按钮
        • date/time/datetime-local：日期/时间/日期时间
        • number：数字输入框
        • email：邮件输入框
        • hidden：定义隐藏域
        • submit/reset/button：提交按钮/重置按钮/可点击按钮
      • value：文本
      • name：元素的名称（变量名）
    • <textarea>：多行文本输入，文本域，大小可自行进行调整
    • <select>：下拉列表
    • <option>：下拉选项
    • <button>：按钮（提交或重置）

其他常见标签#

  • <script>：引入或书写JavaScript脚本
  • <link>：引入或书写CSS文件
  • <meta>：网页信息，如字符集、关键词

CSS样式控制#

CSS的引入方式#

  1. 内联样式（Inline Style） 行内样式：写在标签的style属性中 示例： 
    <p style="color: red; font-size: 16px;">这是红色文字</p>
  2. 内部样式表（Internal Style Sheet） 写在<style>标签中（可以写在任何位置，但通常写在<head>标签中） 
    <head>
  <style>
    p {
      color: blue;
      font-size: 18px;
    }
    h1 {
     color: red;
    }
  </style>
</head>
  3. 外部样式表（External Style Sheet） 写在一个单独的.css文件中（需要通过link标签在网页中引入 示例： 
    <head>
  <link rel="stylesheet" href="style.css">
</head>
    • rel:描述与当前文档的关系
      • stylesheet: 当前文档的样式表
    • href:资源的URL地址

CSS选择器#

用来选取需要设置样式的元素

基本选择器#

类型 示例 说明
元素选择器 p { color: red; } 选中所有 <p> 段落元素
类选择器 .title { color: blue; } 选中 class="title" 的元素
ID 选择器 #main { color: green; } 选中 id="main" 的唯一元素
通配选择器 * { margin: 0; } 选中页面中所有元素
示例： 
<p class="text">段落文字</p>
<div id="box">盒子内容</div>
<style>
  p { color: red; }
  .text { font-size: 18px; }
  #box { background-color: yellow; }
</style>

层级选择器（关系选择器）#

类型 示例 含义
后代选择器 div p 选中所有位于 <div> 内部的 <p> 元素
子代选择器 div > p 选中直接子元素 <p>
相邻兄弟选择器 h1 + p 选中紧跟在 <h1> 后的第一个 <p>
通用兄弟选择器 h1 ~ p 选中与 <h1> 同级、在其后的所有 <p>
示例： 
<div>
  <p>我是div里的p</p>
</div>
<p>我是外面的p</p>
<style>
  div p { color: red; }    /* 只选中第一个p */
</style>

属性选择器#

选择器 示例 含义
[type] input[type] 选中所有有 type 属性的 input
[type="text"] input[type="text"] 选中 type="text" 的输入框
[title~="news"] [title~="news"] 选中属性中包含单词“news”的元素
[href^="https"] a[href^="https"] 选中以 https 开头的链接
[href$=".jpg"] a[href$=".jpg"] 选中以 .jpg 结尾的链接
[href*="blog"] a[href*="blog"] 选中 href 中包含 blog 的链接
示例： 
<input type="text">
<input type="password">
<style>
  input[type="text"] { background-color: yellow; }
</style>

伪类选择器#

伪类用于选择元素的特殊状态（如鼠标悬停、已访问链接、首个子元素等）。

伪类 示例 含义
:hover a:hover 鼠标悬停时样式
:focus input:focus 输入框获得焦点时样式
:first-child p:first-child 第一个子元素
:last-child p:last-child 最后一个子元素
:nth-child(n) li:nth-child(2) 第 n 个子元素
示例： 
<a href="#">鼠标悬停试试</a>
<style>
  a:hover { color: red; text-decoration: underline; }
</style>

伪元素选择器#

伪元素用于选中元素内容的某个部分。

伪元素 示例 含义
::before p::before 在元素内容前插入内容
::after p::after 在元素内容后插入内容
::first-line p::first-line 选中文本第一行
::first-letter p::first-letter 选中文本第一个字母
示例： 
<p>欢迎学习 JavaWeb！</p>
<style>
  p::before { content: "👉 "; color: red; }
  p::after { content: " ✅"; color: green; }
</style>

CSS层叠与优先级#

CSS 的名字叫“层叠样式表”，意味着同一元素可能有多个样式规则，最终显示哪一个由 优先级（Specificity） 决定。 优先级从高到低：

  1. 内联样式 >
  2. 内部样式/外部样式（谁后加载谁优先级高
  3. ID 选择器 #id >
  4. 类选择器 .class、属性选择器 [type=“text”] >
  5. 标签选择器 p, div >
  6. 浏览器默认样式 选择器优先级总结
    选择器类型 优先级高低 示例
    内联样式 最高 <p style="color:red;">
    ID 选择器 次高 #id
    类、属性、伪类选择器 中等 .class, [attr], :hover
    元素、伪元素选择器 较低 p, ::before
    通配符 * 最低 * {}
