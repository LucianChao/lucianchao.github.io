Vue#

什么是Vue？#

VUE 是一款用于 构建用户界面 的 渐进式 的JavaScript 框架

  • 官网：https://cn.vuejs.org/

VUE 是一个框架， 也是一个生态

  • 框架：就是一套完整的项目解决方案，用于快速构建项目。
  • 优点：大大提升前端项目的开发效率。
  • 缺点：需要理解记忆框架规则。（参照官网或文档）

    • 引入Vue模块（官方提供）

Vue常用指令#

  • Vue指令：HTML标签上带有 v-前缀的特殊属性，不同的指令具有不同的含义，可以实现不同的功能
  • 常用指令：
    指令 作用
    v-for 列表渲染，遍历容易的元素或者对象的属性
    v-bind 为HTML标签绑定属性值，如设置 href，css样式等
    v-if/v-else-if/v-else 条件性的渲染某元素，判断为true时渲染，否则不渲染
    v-show 根据条件展示某元素，区别在于切换的是display的值
    v-model 在表单元素上创建双向数据绑定
    v-on 为HTML标签绑定事件

v-for#

  • 作用：列表渲染，遍历容易的元素或者对象的属性
  • 语法： 
    <div v-for = "(item, index) in items"></div>
    • 参数说明：
      • items 遍历的数组
      • item 遍历取出的元素
      • index 为索引/下标，从0开始；可以省略，省略index语法：`v-for = “item in items”;
    • 示例： 
      <p v-for="content in contents">{{content}}</p>

信息

注意：遍历的数组，必须在data中定义；想让哪个标签循环多次，就在哪个标签上使用v-for指令

v-for的key#

  • 作用：给元素添加的唯一标识符，便于vue进行列表项的正确排序复用
  • 语法： 
    <div v-for = "(item, index) in items" :key="唯一值">
  • 注意点：
    • key的值只能是 字符串 或 数字类型
    • key的值必须具有 唯一性
    • 推荐使用id作为key（唯一），不推荐使用index作为key（会变化，不对应）
  • 写法： 
    <p v-for="content in contents" :key="content.id">{{content}}</p>

提示

官方推荐在使用 v-for 时提供一个 key 属性，以便可以追踪每个节点，提升渲染性能

示例

<tr v-for="(user, index) in userList" ::key="user.id">
    <td>{{index}}</td>
    <td>{{user.name}}</td>
    <td><img src="{{user.image}}"></td>
    <td>{{user.gender}}</td>
    <td>{{user.job}}</td>
    <td>{{user.entrydate}}</td>
    <td>{{user.updatetime}}</td>
</tr>

v-bind#

  • 作用：动态为HTML标签绑定属性值，如设置href, src, style样式等。
  • 语法： 
    <div v-bind:属性名="属性值">
    简化： 
    <div :属性名="属性值">

在上方示例中<img src="{{user.image}}">并不正确，""中无法使用插值表达式，想要绑定属性就需要用v-bind了

正确的方法应该是：

    <td><img :src="user.image"></td>
    <!-- 简写方式 -> : -->
     <!-- 去掉{{}}，直接给值 -->

信息

v-bind所绑定的数据，必须在data中定义。

示例：

  <head>

      <a href="https://www.steamedfish.top/">链接1</a>

v-if & v-show#

  • 作用：这两类指令都是用来 控制元素的显示与隐藏的
  • v-if：
    • 语法：v-if = "表达式"，表达式值为true，显示；false，隐藏
    • 原理：基于 条件判断，控制创建或移除元素节点（条件渲染）
    • 应用场景： 要么显示，要么不显示，不频繁切换的场景
    • 其他：可以配合 v-else-if/v-else 进行链式调用条件判断
  • v-show：
    • 语法：v-show = "表达式"，表达式值为true，显示；false，隐藏
    • 原理：基于CSS样式display来控制显示与隐藏
    • 场景：频繁切换显示隐藏的场景

案例#

在前面的用户列表页面中

<td>{{user.gender}}</td>
<td>{{user.job}}</td>

这一行代码的渲染效果只是user.gender与user.job的值，而我们希望它能够根据值来进行逻辑判断，区分用户性别与职位

<td>
  <span v-if="user.gender == 1">男</span>
  <span v-else-if="user.gender == 2">女</span>
  <span v-else>武装直升机</span>
</td>
<td>
  <span v-show="user.job == 1">讲师</span>
  <span v-show="user.job == 2">班主任</span>
  <span v-show="user.job != 1 && user.job != 2">其他</span>
</td>

信息

可以很轻易的发现，v-show与v-if不同，它不能与v-else配合形成复合逻辑
v-else-if必须出现在v-if之后，可以出现多个；
v-else必须出现在v-if/v-else-if之后。

v-model#

  • 作用：在表单元素上使用， 双向数据绑定。可以方便的 获取 或 设置 表单项数据
  • 语法：v-model = "变量名"

信息

v-model中绑定的变量，必须在data中定义。

示例:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>v-model</title>
  </head>
  <body>
    <div id="app"><input type="text" v-model="name" />{{name}}</div>
    <script type="module">
      import { createApp } from "https://unpkg.com/vue@3/dist/vue.esm-browser.js";
      createApp({
        data() {
          return {
            name: "Tom",
          };
        },
      }).mount("#app");
    </script>
  </body>
</html>

在此示例中，Tom字段在网页表单中进行更改时，后端的name变量随之一起更改，因为前端后端是双向绑定的

v-on#

  • 作用：为html标签绑定事件（添加事件监听）
  • 语法：
    • v-on:事件名 = "内联语句"
    • v-on:事件名 = "函数名"
    • 简写为： @事件名 = "..."

在我们之前学习过的JavaScript中有学到过事件监听

事件源.addEventListenner('事件类型', 事件触发执行的函数);

<body>
  <input type="button" value="点我一下试试" id="btn1">
  <script>
    // 事件监听   --可以绑定多个事件
    document.querySelector("#btn1").addEventListener('click',()=>{
      console.log('试试就试试')
    });
  </script>
</body>

现在我们用v-on的 内联语句 重写这段代码

<div id="app">
    <input type="button" value="点我一下试试"
    v-on:click="console.log('试试就试试');">
</div>

注释

现在这段代码变得简单、清晰、明了，大大提高了开发效率

我们一起来思考一个问题
数据从哪来？

在上面的例子中，我们在data中的userList字段写入了用户列表数据，它是写死的
而在实际开发中，数据往往需要从服务器请求才能得到

该如何请求数据并进行异步处理呢？

Ajax#

Ajax——全称（ Asynchronous JavaScript And XML）异步的JavaScript和

XML：（Extensible Markup Language）可扩展标记语言，本质是一种数据格式，可以用来存储复杂的数据结构。
尽管Ajsx中的X代表XML，但是Json才是首选，因为它更加轻量。

  • 作用：
    • 数据交换：通过Ajax可以给服务器发送请求，并获取服务器响应的数据。
    • 异步交互：可以在不重新加载整个页面的情况下，与服务器交换数据并更新部分网页的技术，如：搜索联想、用户名是否可用的校验等等

同步和异步#

发送一个请求后，不需要等待返回，随时可用再发送下一个请求，即不需要等待

原生Ajax(了解即可)#

  1. 创建XMLHttpRequest对象：用于和服务器交换数据（https://mock.apifox.cn/m1/3128855-0-default/emp/list）
  2. 向服务器发送请求
  3. 获取服务器响应数据
<button id="ajaxButton" type="button">发送请求</button>
<script>
  document.querySelector("#ajaxButton").addEventListener("click",()=>{
    // 1.创建XMLHttpRequest对象
    const httpRequest = new XMLHttpRequest();
    // 2.向服务器发送异步请求
    httpRequest.open("GET", "https://mock.apifox.cn/m1/3128855-0-default/emp/list");
    httpRequest.send();
    // 3.获取服务器响应数据
    httpRequest.onreadystatechange = function () {
      if (httpRequest.readyState === 4 && httpRequest.status === 200){
        console.log(httpRequest.responseText);
      }
    };
  });
</script>

Axios#

对原生的Ajax进行了封装，简化书写，快速开发

官网https://www.axios-http.cn/

Axios的使用步骤#

  • 引入Axios的js文件（参照官网）
  • 使用Axios发送请求，并获取相应结果 
    <script scr="https://unpkg.com/axios/dist/axios.min.js"></script>

    axios({
  // 请求方式
  methods:'GET',
  // 请求地址（后端服务地址）
  url:'https://mock.apifox.cn/m1/3083103-0-default/emps/list'
}).then((result) => {
  // 成功之后的回调函数   拿到封装的结果并打印
  console.log(result.data);
}).catch((err) => {
  // 错误信息
  alert(err);
})

提示

提示：在使用axios时，在axios之后，输入 thenc会自动生成成功及失败回调函数结构

Axios-请求方式别名#

为了方便起见，Axios已经为所有支持的请求方法提供了别名

格式：axios.请求方式(url [,data [, config]])

axios.get("https://mock.apifox.cn/m1/3083103-0-default/emps/list")
.then((result) => {
  console.log(result.data);
})
.catch((err) => {
  alert(err);

生命周期#

  • 生命周期：指一个对象从创建到销毁的整个过程。
  • 生命周期的八个阶段：每触发一个生命周期事件，会 自动 执行一个生命周期方法（钩子/钩子方法/钩子函数）
    状态 阶段周期
    beforeCreate 创建前
    created 创建后
    beforeMount 载入前
    mounted 挂载完成
    beforeUpdate 数据更新前
    beforeUnmount 组件销毁前
    Unmounted 组件销毁后 
<script type="module">
  import { createApp } from "https://unpkg.com/vue@3/dist/vue.esm-browser.js";

  createApp({
    data() {
      return {
        message:"Hello Vue"
      };
    },
    methods: {
      // 省略部分代码
    },
    // 生命周期-钩子函数 mounted
    mounted() {
      console.log('Vue挂载完成，发送请求获取数据...')
    }
  }).mount("#app");
</script>

案例#

优化用户列表，在页面加载完成后，就执行一次查询函数search，让页面在未点击查询按钮前也能展示数据

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>列表渲染</title>
  </head>
  <body>
    <div id="app">
      <div id="center">
        姓名：<input type="text" name="name" v-model="name" />
        性别：
        <select name="gender" v-model="gender">
          <option value="1">男</option>
          <option value="2">女</option>
        </select>
        职位：
        <select name="job" v-model="job">
          <option value="1">讲师</option>
          <option value="2">班主任</option>
          <option value="3">其他</option>
        </select>
        <input class="btn" type="button" value="查询" @click="search" />
      </div>

      <table>
        <tr>
          <th>序号</th>
          <th>姓名</th>
          <th>头像</th>
          <th>性别</th>
          <th>职位</th>
          <th>入职时间</th>
          <th>更新时间</th>
        </tr>
        <tr v-for="(user, index) in userList" ::key="user.id">
          <td>{{index}}</td>
          <td>{{user.name}}</td>
          <td><img :src="user.image" /></td>
          <td>
            <span v-if="user.gender == 1">男</span>
            <span v-else-if="user.gender == 2">女</span>
            <span v-else>武装直升机</span>
          </td>
          <td>
            <span v-show="user.job == 1">讲师</span>
            <span v-show="user.job == 2">班主任</span>
            <span v-show="user.job != 1 && user.job != 2">其他</span>
          </td>
          <td>{{user.entrydate}}</td>
          <td>{{user.updatetime}}</td>
        </tr>
      </table>
    </div>
    <!-- 引入axios -->
    <script src="https://unpkg.com/axios/dist/axios.min.js"></script>
    <script type="module">
      import { createApp } from "https://unpkg.com/vue@3/dist/vue.esm-browser.js";

      createApp({
        data() {
          return {
            name: "",
            gender: "",
            job: "",
            userList: [],
          };
        },
        methods: {
          search() {
            // 输出搜索框中的数据
            console.log(
              `姓名：${this.name}，性别：${this.gender}，职位：${this.job}`
            );
            // 通过axios发送异步请求，动态获取数据 https://web-server.itheima.net/emps/list
            axios
              .get(
                `https://web-server.itheima.net/emps/list?name=${this.name}&gender=${this.gender}&job=${this.job}`
              )
              .then((result) => {
                // 拿到数据
                this.userList = result.data.data;
              })
              .catch((err) => {});
          },
        },
        mounted() {
          // 页面加载完成时，进行数据查询
          this.search();
        },
      }).mount("#app");
    </script>
  </body>
</html>
