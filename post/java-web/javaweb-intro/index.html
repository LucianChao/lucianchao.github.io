何为Web?#

Web，全称World Wide Web（万维网），是一个能够通过浏览器访问的网站系统。 它依赖于 HTTP 协议 来进行通信，在客户端与服务器之间传递数据，通过 URL（统一资源定位符）访问资源。 例如网页、图片、视频、文件等。

Web标准#

Web 标准（Web Standards） 是由国际组织（主要是 W3C：万维网联盟）制定的一系列技术规范，用来确保网页在不同浏览器、不同平台上都能被正确解析和显示。

Web 标准的目标是：

“让 Web 更加开放、统一、可访问。”

一、Web 标准的三大核心组成#

结构（Structure）#

由 HTML（超文本标记语言） 定义。
用来描述网页的内容与语义，例如段落、标题、列表、图片等。
示例：

<h1>欢迎来到我的网站</h1>
<p>这是一个使用标准 HTML 编写的网页。</p>

什么是HTML？#

  • HTML(HyperText Markup Language):超文本标记语言
    • 超文本：超越了文本的限制，比普通文本更强大。除了文字信息，还可以定义图片、音频、视频等内容。
    • 标记语言：由标签"<标签名>“构成的语言
      • HTML标签都是预定义好的。例如：使用<h1>展示标题，使用<img>展示图片，使用<video>展示视频
      • HTML代码直接在浏览器中运行，HTML标签由浏览器解析、

这里给到一个文档网站，方便查阅学习

w3school

表现（Presentation）#

由 CSS（层叠样式表） 实现。
控制网页的外观样式，如字体、颜色、布局。
示例：

h1 {
  color: #0066cc;
  text-align: center;
}

什么是CSS？#

  • CSS(Cascading Style Sheet)：层叠样式表，用于控制页面的样式（表现）

行为（Behavior）#

由 JavaScript（JS） 实现。
用来控制网页的交互效果与动态逻辑。
示例：

alert("欢迎访问我的网页！");

这三者共同构成了 Web 页面的完整表现层次：

HTML 负责结构，CSS 负责样式，JavaScript 负责行为。

二、为什么要遵循 Web 标准#

  1. 跨浏览器兼容性更好

按标准编写的网页在 Chrome、Edge、Firefox、Safari 等浏览器上都能正常显示。

  1. 便于维护与升级

结构、样式、逻辑分离，修改样式或脚本不会影响网页内容。

  1. 利于搜索引擎优化（SEO）

语义化 HTML 有助于搜索引擎理解网页结构，提高排名。

  1. 提高网站可访问性（Accessibility）

对残障人士使用辅助设备（如读屏器）更友好。

Web的组成结构#

一个完整的 Web 系统通常包含三部分：

  • 客户端/前端（Client）
    也就是用户使用的浏览器（如 Chrome、Firefox）。
    负责向服务器发送请求，并展示返回的网页内容。

  • 服务器/后端（Server）
    负责处理客户端的请求，执行相关逻辑（如数据库查询、计算），并返回响应。
    常见的 Web 服务器软件有 Apache、Nginx、Tomcat 等。

  • 数据库（Database）
    用于存储网站数据，如用户信息、商品、评论、订单等。
    常见的数据库有 MySQL、PostgreSQL、Oracle 等。

这三者共同组成了 Web 系统的基础架构：
客户端发出请求 → 服务器处理逻辑 → 数据库存取数据 → 返回响应给客户端。

Web的工作原理#

Web 的核心通信机制是 HTTP（超文本传输协议）。 工作流程如下：

  1. 用户在浏览器输入网址（URL）；
  2. 浏览器向服务器发送 HTTP 请求；
  3. 服务器处理请求（可能访问数据库或执行逻辑）；
  4. 服务器返回 HTTP 响应（通常是 HTML、JSON 或文件）；
  5. 浏览器解析响应并呈现页面。

🔁 这就是常说的 请求—响应模型（Request–Response Model）。

Java Web 技术简介#

Java Web 指的是使用 Java 技术 开发 Web 应用程序的技术体系。 它以 Servlet/JSP 为核心，运行在 Web 容器（如 Tomcat） 中。

Java Web 技术的主要组成包括：

模块 说明
Servlet Java Web 的核心，负责接收请求与生成响应。
JSP（Java Server Pages） 类似 HTML，但可以嵌入 Java 代码，用于动态生成页面。
JDBC Java 数据库连接技术，用于访问数据库。
EL 表达式与 JSTL 标签 简化 JSP 页面逻辑。
Filter 与 Listener 用于请求过滤与事件监听。

随着技术发展，现代 Java Web 已逐步演化为使用 Spring MVC、Spring Boot 等框架的形式，大大提升了开发效率与可维护性。
