Java程序操作数据库#

JDBC#

概述#

  • JDBC: (Java DataBase Connectivity)，就是使用Java语言操作关系型数据库的一套API。
  • 本质：
    • sun公司官方定义的一套操作所有关系型数据库的规范，即接口
    • 各个数据库厂商去实现这套接口，提供数据库驱动jar包
    • 我们可以使用这套接口（JDBC）编程，真正执行的代码时驱动jar包中的实现类

JDBC快速入门#

通过JDBC程序，执行update语句，更新用户表中的数据#

  • 准备工作
    1. 创建项目，引入mysql的驱动、junit依赖
    2. 注册驱动
    3. 获取连接对象 Connection
    4. 获取SQL语句执行对象 Statement
  • 执行SQL语句 
    package com.javaweb;

import org.junit.jupiter.api.Test;

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.Statement;

/**
 * JDBC测试
 */
public class JDBCTest {
    @Test
    public void testUpdate() throws Exception {
        //1.准备工作
        //1.1  加载驱动
        Class.forName("com.mysql.cj.jdbc.Driver");//固定的

        //1.2 获取数据库的连接对象 Connection
        Connection connection = DriverManager.getConnection("jdbc:mysql://localhost:3306/javaweb_db02", "root", "20030420");

        //1.3 获取执行sql语句的对象 Statement
        Statement statement = connection.createStatement();

        //2.  执行sql语句
        //修改用户表id为1的密码，修改为12345678
        int i = statement.executeUpdate("update user set password ='12345678' where id = 1");
        System.out.println(i);
        //3.释放资源
        statement.close();
        connection.close();
    }
}

API详解#

DriverManager#

  • DriverManager（驱动管理器）作用：
    1. 注册驱动：DriverManager.registerDriver()
      • 当加载Driver驱动类时，会自动运行静态代码块中注册驱动的代码
      • Class.forName这部操作可以省略
    2. 获取数据库连接：DriverManger.getConnection(url,user,password)
      • url：数据库连接url
        • 语法：jbdc:mysql://ip地址(域名):端口号/数据库名?参数键值对1&参数键值对2
        • 说明：如果连接的是本地的默认端口的mysql，url可以简写为：jdbc:mysql:///数据库名?参数键值对...
      • user：用户名
      • password：密码

信息

SPI机制：Service Provider Interface，JDK内置的一种服务提供发现机制，可以轻松拓展你的程序（切换实现），实现接口与实现类之间的解耦。

Connection & Statement#

  • Connection的作用：获取执行SQL的对象
    • 执行普通SQL对象Statement：connection.createStatement()
    • 执行预编译SQL对象PreparedStatement：connection.prepareStatement()
  • Statement的作用：执行SQL
    • 执行DDL、DML语句：executeUpdate(sql); 如果时执行DML语句完毕，返回值int代表DML语句影响的行数。
    • 执行DQL语句：executeQuery(sql);：返回值为ResultSet，里面封装了查询结果

ResultSet#

  • 需求：请根据用户名和密码查询用户信息，来模拟用户登录操作
  • ResultSet（结果集对象）：封装了DQL查询语句查询的结果
    • next()：将光标从当前位置向前移动一行，并判断当前行是否为有效行，返回值为boolean
      • true：有效行，当前行有数据
      • false：无效行，当前行无数据
    • getXxx(...)：获取数据，可以根据列的编号获取，也可以根据列名获取（推荐）
@ParameterizedTest
@CsvSource({"xiaoqiao,123456","zhangsan,123456"})
public void testSelect(String uname, String pwd) throws Exception {
    //1.准备工作
    //1.1  加载驱动
    Class.forName("com.mysql.cj.jdbc.Driver");//固定的

    //1.2 获取数据库的连接对象 Connection
    Connection connection = DriverManager.getConnection("jdbc:mysql://localhost:3306/javaweb_db02", "root", "20030420");

    //1.3 获取执行sql语句的对象 Statement
    Statement statement = connection.createStatement();

    //2.  执行sql语句
    //需求：根据用户名和密码查询用户信息，模拟登录功能
    ResultSet resultSet = statement.executeQuery("select * from user where username='" + uname + "' and password='" + pwd + "'");
    //判断是否有值
    while (resultSet.next()) {
        //输出结果--解析ResultSet
//            int id = resultSet.getInt(1);     //下标从1开始
        int id = resultSet.getInt("id");
        String username = resultSet.getString("username");
        String password = resultSet.getString("password");
        String name = resultSet.getString("name");
        int age = resultSet.getInt("age");
        User user = new User(id, username, password, name, age);
        System.out.println(user);
    }

    //3.释放资源
    statement.close();
    connection.close();
}

PreparedStatement#

  • 作用：预编译SQL语句并执行，可以防止SQL注入问题
  • SQL注入：通过控制输入来修改事先定义好的SQL语句，以达到执行代码对服务器进行攻击的方法
  • 通过PreparedStatement解决SQL注入问题：
    1. 获取PreparedStatement对象
    2. 设置参数值
    3. 执行SQL 
      //获取预编译SQL执行对象PreparedStatement  - SQL语句中的参数值，使用？占位符替代
PreparedStatement ps = conn.prapareStatement("select * from emo where username = ? and password = ?");
//设置参数
ps.setString(1, uname);//设置第一个？占位符的值
ps.setString(2, pass); //设置第二个?占位符的值

//执行SQL语句
ResultSet resultSet = ps.executeQuery();
  • 预编译SQL的优势：
    1. 安全（防止SQL注入）
    2. 性能更高

示例：

/**
* 预编译SQL 解决SQL注入漏洞
*
* @param uname
* @param pwd
* @throws Exception
*/
@ParameterizedTest
@CsvSource({"xiaoqiao,123456", "zhangsan,123456"})
public void testSelect2(String uname, String pwd) throws Exception {
  //1.准备工作
  //1.1  加载驱动
  Class.forName("com.mysql.cj.jdbc.Driver");//固定的

  //1.2 获取数据库的连接对象 Connection
  Connection connection = DriverManager.getConnection("jdbc:mysql://localhost:3306/javaweb_db02", "root", "20030420");

  //1.3 获取执行sql语句的对象 Statement
//        Statement statement = connection.createStatement();
  PreparedStatement ps = connection.prepareStatement("select * from user where username = ? and password = ?");
  //1.4 设置参数值
  ps.setString(1, uname);
  ps.setString(2, pwd);
  //2.  执行sql语句
  //需求：根据用户名和密码查询用户信息，模拟登录功能
  ResultSet resultSet = ps.executeQuery();
  //判断是否有值
  while (resultSet.next()) {
      //输出结果--解析ResultSet
//            int id = resultSet.getInt(1);     //下标从1开始
      int id = resultSet.getInt("id");
      String username = resultSet.getString("username");
      String password = resultSet.getString("password");
      String name = resultSet.getString("name");
      int age = resultSet.getInt("age");
      User user = new User(id, username, password, name, age);
      System.out.println(user);
  }

  //3.释放资源
  ps.close();
  connection.close();
}
