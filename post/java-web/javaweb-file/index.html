项目部署#

  1. 把前端工程文件夹中的压缩包，拷贝到一个没有中文，不带空格的目录下，解压。
  2. 安装nvm-weindows
  3. 在 文件资源管理器 路径栏 输入 cmd 按下 Enter，打开 cmd命令行
  4. 在命令行输入如下命令 
    nvm install 18.18.0
nvm use 18.18.0
node -v
  5. 命令行输入，安装环境并启动服务 
    npm install
npm run dev
  6. 启动后端程序后访问前端
