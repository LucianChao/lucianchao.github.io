什么是JavaScript#

  • JavaScript （简称：JS）是一门跨平台、面对对象的脚本语言，用来实现页面的动态效果、交互逻辑和数据处理。它与 HTML、CSS 共同构成 Web 三大核心技术。
  • JavaScript和Java是完全不同的语言，不论是概念还是设计，但是基础语法类似

JavaScript的组成#

  • ECMAScript：规定了JS基础语法核心知识，包括变量、数据类型、流程控制、函数、对象等
  • BOM：浏览器对象模型，用于操作浏览器本身，如：页面弹窗、地址栏操作、关闭窗口等
  • DOM：文档对象模型，用于操作HTML文档，如：改变标签内的内容、改变标签内字体样式等

信息

ECMA：ECMA国际（前身为欧洲计算机制造商协会），制定了标准化的脚本程序设计语言 ECMAScript，这种语言得到广泛应用。而JavaScript是遵守ECMAScript的标准的。

JS引入方式#

内部脚本（写在 HTML 中）#

  • JavaScript代码必须位于<script></script>标签之间
  • 在HTML文档中，可以在任意地方，防止任意数量的<script>
  • 一般会把脚本置于<body>元素的底部，可改善显示速度 示例：
<body>
    <!-- 页面内容 -->
    <script>
        console.log("Hello JavaScript!");
    </script>
</body>

外部脚本（推荐）#

将JS代码定义在外部JS文件中，然后引入到HTML页面中

<script src="js/main.js"></script>

所有 JS 推荐单独存放，便于维护和复用。

JS基础语法#

书写规范#

  • 结束符;：每行结尾以分号结束，结尾的分号可有可无
  • 区分大小写：与Java一样，变量名，函数名以及其他一切东西都是区分大小写的
  • 注释：单行注释、多行注释
  • {}：大括号表示代码块

输出语句#

  • document.write()：向HTML的body内输出内容
  • windows.alert()：弹出警告框
  • console.log()：写入浏览器控制台（开发调试）

变量#

  • JavaScript 中用 let 关键字来声明变量
  • JavaScript 是一门 弱类型 语言，变量可以存放不同类型的值
  • 变量名需要遵守以下规则：
    • 只能用 字母、数字、下划线_、美元符号$组成，且数字不能开头
    • 变量名严格区分大小写
    • 不能使用关键字命名变量，如：let、var、if、for等

变量声明#

let a = 10;   // 可变变量（推荐）
const b = 20; // 常量（推荐）
var c = 30;   // 老写法（不推荐）

推荐规则

  • let：常用变量
  • const：永不改变的值（不能重新赋值）
  • var：尽量不用，作用域混乱

数据类型#

JavaScript的数据类型分为 基本数据类型 和 引用数据类型（对象） JavaScript的基本类型包括：

  • number：数字（整数、小数、NaN(Not a Number)）
  • boolean：布尔。true,false
  • null：空对象。JavaScript是大小写敏感的，因此null、Null、NULL是完全不同的
  • undefined：当声明的变量未初始化时，该变量的默认值是undefined
  • string：字符串，单引号、双引号、反引号皆可，推荐使用 单引号
    • 拼接字符串
    • ` `
    • 内容拼接变量时，使用 ${} 包住变量
    <script>
    let name = 'Tom';
    let age = 18;
    console.log('大家好，我是新入职的' + name + '，今年' + age + '岁，请多多关照');
    console.log(`大家好，我是新入职的${name}，今年${age}岁，请多多关照`);

</script>

运算符#

  • 算术运算符：+, -, *, /, %, ++, --
  • 赋值运算符：=, +=, -=, *=, /=, %=
  • 比较运算符：>, <, >=, <=, !=, ==, ===严格相等
    • =, ==, ===区别：
      • =：赋值
      • ==：在比较时，判断两边的值是否相等（会进行类型转换）
      • ===：全等，比较时，判断两边的类型和值是否都相等（不会进行类型转换）
      • console.log(5 === "5"); // false，类型不同
console.log(5 == "5");  // true，被转换了，强烈不推荐
  • 逻辑运算符：&&, ||, !（与、或、非）
  • 三元运算符：条件表达式 ? true_value : false_value

类型转换#

在某些运算被执行时，系统内部会自动将数据类型进行转换，这种转换称为 隐式转换

  • 字符串 <-> 数字：
    • +号两边只要有一个是字符串，都会把另外一个转成字符串
    • 除了+之外的运算符，比如：-, *, / 都会把字符串转成数字类型
    • 将字符串转为数字类型，还可以使用parseInt("12")这样的函数
    • <script>
    parseInt('A123')    //NaN，不是数字
</script>
  • 其他类型 <-> 布尔类型：
    • number：0 和 NaN 为false，其他均转为true
    • string：空字符串 ('')为false，其他均转为true
    • null 和 undefined ：均转为false

函数#

函数( function )是被设计用来执行特定任务的代码块，方便程序的封装复用。

提示

参考Java的方法，两者可以说是同一种东西

函数的定义#

JavaScript中的函数通过function关键字进行定义

语法：

function functionName(参数1,参数2...){
    //要执行的代码
}

注释

由于JS是一门弱类型语言，形参、返回值都不需要指定类型。

在调用函数时，实参个数与形参个数可以不一致，但建议一致

示例：

function add(a, b) {
    return a + b;
}

函数的调用#

函数名称(实际参数列表)

示例：

let result1 = add(10, 20);
console.log(result1);    //30
let result2 = add(10);     
console.log(result2);    //NaN， b未赋值，-> undifined ， a+b =NaN
let result3 = add(10, 20, 30); 
console.log(result3);    //30，参数列表的30是多余的，不参与计算

匿名函数#

匿名函数是指一种没有名称的函数，由于它们没有名称，因此无法直接通过函数名来调用，而是通过变量或表达式来调用

函数表达式#

let fun = function(a, b) {
    return a + b;
}
console.log(fun(10, 20));   //30

箭头函数（推荐）#

let add = (a, b) => a + b;
console.log(add(10, 20));   //30

立即执行函数#

匿名函数定义后，可以通过变量名直接调用。也可以在定义后立即执行，这称之为 立即执行函数

前半部分为匿名函数，后面直接加上(参数...)可以立即执行一次

(function (a, b) {console.log(a + b);})(10, 20);

示例：

((a, b) => {console.log(a + b);})(10, 20);  //30

JS对象#

内置对象#

Array#

JavaScript中的Array对象用于定义数组

定义#
let 变量名 = new Array(元素列表);
let 变量名 =[元素列表];

示例：

let arr = new Array(1,2,3,4,5);
let arr =[1,2,3,4,5];
访问#
arr[索引] = 值;

示例：

arr[10] = 'hello';

注释

JavaScript中的数组相当于Java中的集合，数组长度是可变的，而JavaScript是弱类型，所以可以存储任意类型的数据

属性#
  • length：设置或返回数组中元素的数量
方法#
  • forEach()：遍历数组中每个有值的元素，并调用一次传入的函数
  • map()：遍历每个数组元素，调用函数进行处理并将结构封装到一个新数组中
  • push()：将一个或多个元素添加到数组的末尾，并返回新的长度
  • pop()：从数组移除最后一个元素，并返回该元素
  • unshift()：在数组开头添加一个或多个元素，并返回数组的新长度
  • shift()：从数组移除第一个元素，并返回该元素
  • splice()：从数组中删除元素
