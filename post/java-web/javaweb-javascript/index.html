什么是JavaScript#

  • JavaScript （简称：JS）是一门跨平台、面对对象的脚本语言，用来实现页面的动态效果、交互逻辑和数据处理。

    它与 HTML、CSS 共同构成 Web 三大核心技术。

  • JavaScript和Java是完全不同的语言，不论是概念还是设计，但是基础语法类似

JavaScript的组成#

  • ECMAScript：规定了JS基础语法核心知识，包括变量、数据类型、流程控制、函数、对象等
  • BOM：浏览器对象模型，用于操作浏览器本身，如：页面弹窗、地址栏操作、关闭窗口等
  • DOM：文档对象模型，用于操作HTML文档，如：改变标签内的内容、改变标签内字体样式等

信息

ECMA：ECMA国际（前身为欧洲计算机制造商协会），制定了标准化的脚本程序设计语言 ECMAScript，这种语言得到广泛应用。而JavaScript是遵守ECMAScript的标准的。

JS引入方式#

内部脚本（写在 HTML 中）#

  • JavaScript代码必须位于<script></script>标签之间
  • 在HTML文档中，可以在任意地方，防止任意数量的<script>
  • 一般会把脚本置于<body>元素的底部，可改善显示速度 示例：
<body>
    <!-- 页面内容 -->
    <script>
        console.log("Hello JavaScript!");
    </script>
</body>

外部脚本（推荐）#

将JS代码定义在外部JS文件中，然后引入到HTML页面中

<script src="js/main.js"></script>

所有 JS 推荐单独存放，便于维护和复用。

JS基础语法#

书写规范#

  • 结束符;：每行结尾以分号结束，结尾的分号可有可无
  • 区分大小写：与Java一样，变量名，函数名以及其他一切东西都是区分大小写的
  • 注释：单行注释、多行注释
  • {}：大括号表示代码块

输出语句#

  • document.write()：向HTML的body内输出内容
  • windows.alert()：弹出警告框
  • console.log()：写入浏览器控制台（开发调试）

变量#

  • JavaScript 中用 let 关键字来声明变量
  • JavaScript 是一门 弱类型 语言，变量可以存放不同类型的值
  • 变量名需要遵守以下规则：
    • 只能用 字母、数字、下划线_、美元符号$组成，且数字不能开头
    • 变量名严格区分大小写
    • 不能使用关键字命名变量，如：let、var、if、for等

变量声明#

let a = 10;   // 可变变量（推荐）
const b = 20; // 常量（推荐）
var c = 30;   // 老写法（不推荐）

推荐规则

  • let：常用变量
  • const：永不改变的值（不能重新赋值）
  • var：尽量不用，作用域混乱

数据类型#

JavaScript的数据类型分为 基本数据类型 和 引用数据类型（对象） JavaScript的基本类型包括：

  • number：数字（整数、小数、NaN(Not a Number)）
  • boolean：布尔。true,false
  • null：空对象。JavaScript是大小写敏感的，因此null、Null、NULL是完全不同的
  • undefined：当声明的变量未初始化时，该变量的默认值是undefined
  • string：字符串，单引号、双引号、反引号皆可，推荐使用 单引号
    • 拼接字符串
    • ` `
    • 内容拼接变量时，使用 ${} 包住变量
    <script>
    let name = 'Tom';
    let age = 18;
    console.log('大家好，我是新入职的' + name + '，今年' + age + '岁，请多多关照');
    console.log(`大家好，我是新入职的${name}，今年${age}岁，请多多关照`);

</script>

使用typeof运算符可以获取数据类型：

运算符#

  • 算术运算符：+, -, *, /, %, ++, --
  • 赋值运算符：=, +=, -=, *=, /=, %=
  • 比较运算符：>, <, >=, <=, !=, ==, ===严格相等
    • =, ==, ===区别：
      • =：赋值
      • ==：在比较时，判断两边的值是否相等（会进行类型转换）
      • ===：全等，比较时，判断两边的类型和值是否都相等（不会进行类型转换）
      • console.log(5 === "5"); // false，类型不同
console.log(5 == "5");  // true，被转换了，强烈不推荐
  • 逻辑运算符：&&, ||, !（与、或、非）
  • 三元运算符：条件表达式 ? true_value : false_value

类型转换#

在某些运算被执行时，系统内部会自动将数据类型进行转换，这种转换称为 隐式转换

  • 字符串 <-> 数字：
    • +号两边只要有一个是字符串，都会把另外一个转成字符串
    • 除了+之外的运算符，比如：-, *, / 都会把字符串转成数字类型
    • 将字符串转为数字类型，还可以使用parseInt("12")这样的函数
    • <script>
    parseInt('A123')    //NaN，不是数字
</script>
  • 其他类型 <-> 布尔类型：
    • number：0 和 NaN 为false，其他均转为true
    • string：空字符串 ('')为false，其他均转为true
    • null 和 undefined ：均转为false

函数#

函数( function )是被设计用来执行特定任务的代码块，方便程序的封装复用。

提示

参考Java的方法，两者可以说是同一种东西

函数的定义#

JavaScript中的函数通过function关键字进行定义

语法：

function functionName(参数1,参数2...){
    //要执行的代码
}

注释

由于JS是一门弱类型语言，形参、返回值都不需要指定类型。

在调用函数时，实参个数与形参个数可以不一致，但建议一致

示例：

function add(a, b) {
    return a + b;
}

函数的调用#

函数名称(实际参数列表)

示例：

let result1 = add(10, 20);
console.log(result1);    //30
let result2 = add(10);     
console.log(result2);    //NaN， b未赋值，-> undifined ， a+b =NaN
let result3 = add(10, 20, 30); 
console.log(result3);    //30，参数列表的30是多余的，不参与计算

匿名函数#

匿名函数是指一种没有名称的函数，由于它们没有名称，因此无法直接通过函数名来调用，而是通过变量或表达式来调用

函数表达式#

let fun = function(a, b) {
    return a + b;
}
console.log(fun(10, 20));   //30

箭头函数（推荐）#

let add = (a, b) => a + b;
console.log(add(10, 20));   //30

立即执行函数#

匿名函数定义后，可以通过变量名直接调用。也可以在定义后立即执行，这称之为 立即执行函数

前半部分为匿名函数，后面直接加上(参数...)可以立即执行一次

(function (a, b) {console.log(a + b);})(10, 20);

示例：

((a, b) => {console.log(a + b);})(10, 20);  //30

Array#

JavaScript中的Array对象用于定义数组

定义#
let 变量名 = new Array(元素列表);
let 变量名 =[元素列表];

示例：

let arr = new Array(1,2,3,4,5);
let arr =[1,2,3,4,5];
访问#
arr[索引] = 值;

示例：

arr[10] = 'hello';

注释

JavaScript中的数组相当于Java中的集合，数组长度是可变的，而JavaScript是弱类型，所以可以存储任意类型的数据

属性#
  • length：设置或返回数组中元素的数量
方法#
  • forEach()：遍历数组中每个有值的元素，并调用一次传入的函数
  • map()：遍历每个数组元素，调用函数进行处理并将结构封装到一个新数组中
  • push()：将一个或多个元素添加到数组的末尾，并返回新的长度
  • pop()：从数组移除最后一个元素，并返回该元素
  • unshift()：在数组开头添加一个或多个元素，并返回数组的新长度
  • shift()：从数组移除第一个元素，并返回该元素
  • splice()：从数组中删除元素

JSON#

概念：JabaScript Object Notatiom，JavaScript对象标记法。
JSON是通过JavaScript对象表记法书写的 文本（本质上是个字符串）

由于其语法简单，层次结构清晰鲜明，现多用于作为 数据载体，在网络中进行数据传输。

定义#
let 变量名 = '{"key1":value1, "key2":value2}';
  • value的数据类型为：
    • 数字（整数或浮点数）
    • 字符串（在""双引号中
    • 逻辑值（true或false）
    • 数组（在[]方括号中）

BOM#

  • 概念：浏览器对象模型（Browser Object Model），允许JS与浏览器对话，JS将浏览器的各个组成部分封装为对象。
  • 组成：
    • Window：浏览器窗口对象
    • Navigator：浏览器对象
    • Screen：屏幕对象
    • History：历史记录对象
    • Location：地址栏对象

信息

这里重点学习Window和Location

Window#

  • 介绍：浏览器窗口对象
  • 获取：直接使用Window，其中Window.可以省略
  • 属性：
    • history：对History对象的只读引用。请参阅History对象
    • location：用于窗口或框架的 Location 对象。请参阅Location对象
  • 方法
    • alert()：显示带有一段消息和一个确认按钮的警告框
    • confirm()：显示一段消息以及确认按钮和取消按钮的对话框
      • 示例：
      // confirm  --确认提示框  --返回一个boolean值
let flag = confirm("您确认要删除吗？");
console.log(flag);
    • setInterval()：按照指定的周期（以毫秒计）来调用函数或计算表达式
      • 示例：
      // setInterval  --周期性的定时器，参数1：要触发的函数，参数2：时间（单位ms）
let i = 0;
setInterval(() => {
  i++;
  console.log("定时器执行" + i + "次");
}, 1000);
    • setTimeout()：在指定的毫秒数后调用函数或计算表达式（只会触发一次）

  • 获取：使用window.location获取，其中window.可以省略
  • 属性：
    • href：设置或返回完整的URL
      • 示例：

DOM#

  • 概念：Document Object Model，文档对象模型
  • DOM对象：浏览器根据HTML标签生成的JS对象
    • 所有标签属性都可以在这个对象上面找到
    • 修改这个对象的属性，就会自动映射到标签身上
    • DOM的核心思想：将网页的内容当做对象来处理
  • 将标记语言的各个组成部分封装为对应的对象：
    • Document：整个文档对象
    • Element：元素对象
    • Attribute：属性对象
    • Text：文本对象
    • Comment：注释对象
  • 通过DOM，就能对HTML进行操作
    • 改变 HTML 元素的内容
    • 改变 HTML 元素的样式(CSS)
    • 对 HTML DOM 事件作出反应
    • 添加和删除 HTML 元素

document对象#

  • 网页中所有内容都封装在document对象中
  • 它提供的属性和方法都是用来访问和操作网页内容的，如：document.write(...)

获取元素对象#

想要访问或者操作DOM对象的属性或方法，就得先获取元素对象

  • 获取匹配到的第一个元素：document.querySelector("选择器");
  • 获取匹配到的所有元素：document.querySelectorAll("选择器");
    • 注意：得到的是一个NodeList节点集合，是一个伪数组，但没有push, pop等数组方法
  • 示例： 
    <span id="sid">DOM元素1</span>
<span class="txt">DOM元素2</span>
<span class="txt">DOM元素3</span>

    let items = document.querySelector("#sid");
let items = document.querySelector(".txt");    //第一个 --> DOM元素2
let items = document.querySelector("span");   //第一个 --> DOM元素1
  • 其他获取方式（了解即可） 
    let ele = document.getElementById("sid");   //根据id获取一个元素
let eles = document.getElementsByTagName("div");    //根据标签名获取一类元素，获取页面所有的 div
let eles = document.getElementsByClassName("cls")； //根据类名(class属性)获取元素，获取页面所有类名为cls的元素

实战案例#

模拟一个支付完成页面跳转首页的功能

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>对象-DOM</title>
    <style>
      .center {
        text-align: center;
        margin-top: 50px;
      }
    </style>
  </head>
  <body>
    <div class="center">
      您好，支付完成，<span id="time">5</span>秒之后将自动跳转至
      <a href="https://www.jd.com">官网 </a>首页~
    </div>
    <script>
      // 需求：进行时间倒计时，倒计时结束后，自动跳转到官网首页
      // 分析：DOM操作、周期性的定时器setInterval、location的地址修改
      // 1.获取dom元素对象
      let span = document.querySelector("#time");
      let timeValue = span.innerHTML;
      // 每过1s，时间减一
      setInterval(() => {
        timeValue--;
        if (timeValue <= 0) {
          /* 当时间到0的时候，跳转页面 */
          location.href = "https://www.jd.com";
        }
        span.innerHTML = timeValue;
      }, 1000);
    </script>
  </body>
</html>

事件监听#

什么是事件？什么是事件监听？#

  • 事件：HTML事件是发生在HTML元素的"事情"。比如：
    • 按钮被点击
    • 鼠标移动到元素上
    • 按下键盘按键
  • 事件监听：JavaScript可以在事件触发时，就立即调用一个函数做出响应，也称为 事件绑定 或 注册事件

事件监听的定义#

  • 语法 
    事件源.addEventListenner('事件类型', 事件触发执行的函数);
    • 早期版本写法（了解一下即可） 
      事件源.on事件属性 = function() {...};
    • 区别： on方式会被覆盖，addEventListener方式可以·绑定多次，拥有更多特性，推荐使用
  • 事件监听三要素
  • 事件源：哪个dom元素触发了事件，要获取dom元素
  • 事件类型：用什么方式触发，比如：click鼠标单击，mouseover鼠标经过
  • 事件触发执行函数：要做什么事
  • 示例： 
    <body>
  <input type="button" value="点我一下试试1" id="btn1">
  <input type="button" value="点我一下试试2" id="btn2">
  <script>
    // 事件监听   --可以绑定多个事件
    document.querySelector("#btn1").addEventListener('click',()=>{
      alert("试试就试试1")
    });
    // 早期写法   --调用事件属性--只能绑定一次，后定义的会覆盖前面的
    document.querySelector("#btn2").onclick = function () {
      alert("试试就试试2");
    };
  </script>
</body>

正则表达式#

  • 介绍：正则表达式（Regular Expression，也简称为正则），定义了字符串组成的规则。

  • 作用：通常用来验证数据格式、查找替换文本 等。

  • 定义：

    1. 正则表达式字面量（不要加引号）
    2. 创建正则对象RegExp
    • 示例： 
      const reg1 = /abc/;     //推荐
const reg2 = new RegExp('abc');

  • 方法：

    text(str)：判断指定字符串是否符合规则，符合返回true，不符合返回false

正则表达式语法#

  • 语法：
    • 普通字符：大多数的字符仅能描述他们本身，这些字符称作 普通字符，比如字母和数字
    • 特殊字符：一些具有特殊含义的字符，极大提高了灵活性和强大的匹配功能
      • ^：表示以什么开始
      • $：表示以什么结束
      • []：表示某个范围内的单个字符，如：[0-9]单个数字字符
      • .：表示任意单个字符，除了换行和行结束符
      • \w：代表单词字符：字母、数字、下划线(_)，相当于[A-Za-z0-9]
      • \d：代表数字字符：相当于[0-9]
      • \s：代表空格（包括换行符、制表符、空格等）
    • 量词：表示要匹配的字符或表达式的数量
      • ?：零个或一个
      • *：零个或多个
      • +：一个或多个（至少一个）
      • {n}：n个
      • {m,}：至少m个
      • {m,n}：至少m个，最多n个
  • 示例： 
    let reg = /\w{4,16}/;   //4-16位单词字符  
alert(reg.test("abc"));   //false
alert(reg.test("abcddd"));//true
alert(reg.test("123abc"));//true
